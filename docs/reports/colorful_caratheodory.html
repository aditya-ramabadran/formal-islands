Informal statement:

Let \(X_1,\dots,X_{d+1}\) be finite subsets of \(\mathbb{R}^d\). Assume that for each \(i\), \(0 \in \operatorname{conv}(X_i)\). Then there exist points \(x_i \in X_i\) for \(i=1,\dots,d+1\) such that \(0 \in \operatorname{conv}\{x_1,\dots,x_{d+1}\}\).

Informal proof:

We prove the theorem by a distance-minimization argument. Because each \(X_i\) is finite, there are only finitely many colorful sets \(S=\{x_1,\dots,x_{d+1}\}\), \(x_i\in X_i\). For each such colorful set \(S\), its convex hull \(\operatorname{conv}(S)\) is compact, so there exists at least one point of \(\operatorname{conv}(S)\) of minimal Euclidean distance from the origin. Among all colorful sets \(S\), choose one for which the distance \(\operatorname{dist}(0,\operatorname{conv}(S))\) is minimal. Fix such a minimizing colorful set \(S^*=\{x_1,\dots,x_{d+1}\}\), and let \(z\in \operatorname{conv}(S^*)\) be a point of minimal norm in \(\operatorname{conv}(S^*)\). If \(z=0\), then \(0\in \operatorname{conv}(S^*)\), and we are done. So suppose for contradiction that \(z\neq 0\). By the Active Vertices Hyperplane Lemma, \(z\) can be represented as a convex combination of at most \(d\) points of \(S^*\). Hence, there is at least one index \(j \in \{1,\dots,d+1\}\) such that \(z \in \operatorname{conv}(S^*\setminus\{x_j\})\). By the Distance Improvement Lemma, since \(0\in \operatorname{conv}(X_j)\), we can find a replacement point \(x_j' \in X_j\) to form a new colorful set \(S' = (S^*\setminus\{x_j\}) \cup \{x_j'\}\) such that \(\operatorname{dist}(0,\operatorname{conv}(S')) < \operatorname{dist}(0,\operatorname{conv}(S^*))\). This contradicts the choice of \(S^*\) as a colorful set minimizing the distance from the origin. The contradiction arose from assuming \(z\neq 0\). Hence necessarily \(z=0\), and \(0 \in \operatorname{conv}(S^*)\). Since \(S^*\) is colorful, this completes the proof.

/-- **Distance Improvement Lemma** (Colorful Carathéodory).

Let `S_rest` be a set of points in `EuclideanSpace ℝ (Fin d)`, and let `z ≠ 0` be a point
in `convexHull ℝ S_rest`. If `X_j` is a set with `0 ∈ convexHull ℝ X_j`, then there exists
a point `x_j' ∈ X_j` and a point `w ∈ convexHull ℝ (S_rest ∪ {x_j'})` with `‖w‖ < ‖z‖`.

This formalizes the key step in Bárány's proof of the Colorful Carathéodory Theorem:
if `z` is the closest point to the origin in `conv(S*)` and `z ∈ conv(S* \ {x_j})`,
replacing `x_j` with a suitable `x_j' ∈ X_j` strictly decreases the distance to the origin. -/

    (S_rest : Set (EuclideanSpace ℝ (Fin d)))
    (X_j : Set (EuclideanSpace ℝ (Fin d)))
    (z : EuclideanSpace ℝ (Fin d))
    (hz_ne : z ≠ 0)
    (hz_conv : z ∈ convexHull ℝ S_rest)
    (h_origin : (0 : EuclideanSpace ℝ (Fin d)) ∈ convexHull ℝ X_j) :
    ∃ x_j' ∈ X_j, ∃ w ∈ convexHull ℝ (S_rest ∪ {x_j'}), ‖w‖ < ‖z‖ := by
  obtain ⟨x_j', hx_j'_mem, hx_j'_inner⟩ :=

    norm_combo_lt_of_inner_nonpos x_j' z hz_ne hx_j'_inner

    combo_mem_convexHull_union S_rest x_j' z t ht_pos.le ht_le hz_conv, ht_norm⟩
