Young's convolution inequality

Young's convolution inequality. Let \(1 \le p,q,r < \infty\) satisfy \[ 1 + \frac1r = \frac1p + \frac1q. \] Let \(f \in L^p(\mathbb{R})\) and \(g \in L^q(\mathbb{R})\), and define their convolution by \[ (f*g)(x) := \int_{\mathbb{R}} f(x-y)g(y)\,dy. \] Then \(f*g \in L^r(\mathbb{R})\) and \[ \|f*g\|_{L^r} \le \|f\|_{L^p}\,\|g\|_{L^q}. \] This is the classical finite-exponent form of Young's convolution inequality on \(\mathbb{R}\).

Informal statement:

If $p=1$ or $q=1$, then Young's convolution inequality holds.

Informal proof:

Suppose $p=1$. Then the relation $1 + \frac1r = 1 + \frac1q$ implies $r=q$. By Minkowski's integral inequality, $\|f*g\|_{L^q_x} = \left\| \int_{\mathbb{R}} f(y) g(x-y)\,dy \right\|_{L^q_x} \le \int_{\mathbb{R}} |f(y)|\, \|g(x-y)\|_{L^q_x}\,dy$. Since translation preserves the $L^q$-norm, $\|g(x-y)\|_{L^q_x} = \|g\|_{L^q}$. Therefore $\|f*g\|_{L^q} \le \int_{\mathbb{R}} |f(y)|\,\|g\|_{L^q}\,dy = \|f\|_{L^1}\,\|g\|_{L^q}$. Since $r=q$, this is exactly $\|f*g\|_{L^r} \le \|f\|_{L^p}\,\|g\|_{L^q}$. The case $q=1$ is completely symmetric, yielding $\|f*g\|_{L^p} \le \|f\|_{L^p}\,\|g\|_{L^1}$.

Informal statement:

Assume $p,q > 1$ and $f,g \ge 0$. Let $s$ and $t$ be defined by $\frac1s := \frac1p - \frac1r$ and $\frac1t := \frac1q - \frac1r$. Then for each $x \in \mathbb{R}$, \[ (f*g)(x) \le \left(\int_{\mathbb{R}} f(x-y)^p g(y)^q\,dy\right)^{1/r} \|f\|_{L^p}^{1-p/r} \|g\|_{L^q}^{1-q/r}. \]

Informal proof:

Since $r \ge p$ and $r \ge q$, $s,t \in [1,\infty]$. We have $\frac1r + \frac1s + \frac1t = \frac1p + \frac1q - \frac1r = 1$. Factor the integrand as $f(x-y)g(y) = \bigl(f(x-y)^{p/r} g(y)^{q/r}\bigr) \cdot f(x-y)^{1-p/r} \cdot g(y)^{1-q/r}$. Apply Hölder's inequality in $y$ with exponents $r,s,t$ to get $(f*g)(x) \le \left(\int_{\mathbb{R}} f(x-y)^p g(y)^q\,dy\right)^{1/r} \left(\int_{\mathbb{R}} f(x-y)^{(1-p/r)s}\,dy\right)^{1/s} \left(\int_{\mathbb{R}} g(y)^{(1-q/r)t}\,dy\right)^{1/t}$. Since $(1-p/r)s = p$ and $(1-q/r)t = q$, the second and third factors become $\left(\int_{\mathbb{R}} f(x-y)^p\,dy\right)^{1/s}$ and $\left(\int_{\mathbb{R}} g(y)^q\,dy\right)^{1/t}$. By translation invariance of the $L^p$-norm, these equal $\|f\|_{L^p}^{p/s}$ and $\|g\|_{L^q}^{q/t}$. Using $p/s = 1-p/r$ and $q/t = 1-q/r$, we obtain the stated pointwise bound.

/-!
# Pointwise Hölder bound for Young's convolution inequality

For measurable `f g : ℝ → ℝ≥0∞` and exponents `p, q > 1`, `r > 0` with `1/p + 1/q = 1 + 1/r`:

  `∫⁻ y, f(x-y) * g(y) ≤ (∫⁻ y, f(x-y)^p * g(y)^q)^(1/r) *
                           (∫⁻ y, f(y)^p)^(1/p - 1/r) *
                           (∫⁻ y, g(y)^q)^(1/q - 1/r)`

The proof uses a three-function Hölder inequality with exponents `r`, `s := 1/(1/p - 1/r)`,
`t := 1/(1/q - 1/r)` satisfying `1/r + 1/s + 1/t = 1`, applied to the factorization
  `f(x-y) * g(y) = [f(x-y)^(p/r) * g(y)^(q/r)] * f(x-y)^(1-p/r) * g(y)^(1-q/r)`.
-/

/-! ## Pointwise Hölder bound -/

/-
**Pointwise Hölder bound for convolution.**

For measurable `f g : ℝ → ℝ≥0∞` and exponents `p, q > 1`, `r > 0` with `1/p + 1/q = 1 + 1/r`,
the convolution integral satisfies:
  `∫⁻ y, f(x-y) * g(y) ≤ (∫⁻ y, f(x-y)^p * g(y)^q)^(1/r) *
                           (∫⁻ y, f(y)^p)^(1/p - 1/r) *
                           (∫⁻ y, g(y)^q)^(1/q - 1/r)`.
-/
theorem pointwise_holder_aristotle
    {p q r : ℝ} (hp : 1 < p) (hq : 1 < q) (hr : 0 < r)
    (hpqr : 1 / p + 1 / q = 1 + 1 / r)
    {f g : ℝ → ℝ≥0∞} (hf : Measurable f) (hg : Measurable g) (x : ℝ) :
    ∫⁻ y, f (x - y) * g y ≤
      (∫⁻ y, f (x - y) ^ p * g y ^ q) ^ (1 / r) *
      (∫⁻ y, f y ^ p) ^ (1 / p - 1 / r) *
      (∫⁻ y, g y ^ q) ^ (1 / q - 1 / r) := by
  -- Set $s$ and $t$ using the equalities $1/s = 1/p - 1/r$ and $1/t = 1/q - 1/r$, so that $s, t > 1$.
  have hs_pos : 0 < 1 / p - 1 / r := by
    ring_nf at *; nlinarith [ inv_pos.mpr hr, inv_pos.mpr ( zero_lt_one.trans hp ), inv_pos.mpr ( zero_lt_one.trans hq ), mul_inv_cancel₀ ( ne_of_gt hr ), mul_inv_cancel₀ ( ne_of_gt ( zero_lt_one.trans hp ) ), mul_inv_cancel₀ ( ne_of_gt ( zero_lt_one.trans hq ) ) ] ;
  have ht_pos : 0 < 1 / q - 1 / r := by
    linarith [ one_div_lt_one_div_of_lt zero_lt_one hp, one_div_lt_one_div_of_lt zero_lt_one hq ]
  set s := 1 / (1 / p - 1 / r)
  set t := 1 / (1 / q - 1 / r);
  -- By the three-function Hölder inequality, we have
  have h_holder : ∫⁻ y, (f (x - y)) ^ (p / r) * (g y) ^ (q / r) * (f (x - y)) ^ (1 - p / r) * (g y) ^ (1 - q / r) ≤
               (∫⁻ y, ((f (x - y)) ^ (p / r) * (g y) ^ (q / r)) ^ r) ^ (1 / r) * (∫⁻ y, (f (x - y)) ^ ((1 - p / r) * s)) ^ (1 / s) *
               (∫⁻ y, (g y) ^ ((1 - q / r) * t)) ^ (1 / t) := by
                 convert lintegral_three_mul_le ( show 1 < r by
                                                   contrapose! hs_pos; ring_nf at *; nlinarith [ inv_pos.2 hr, mul_inv_cancel₀ hr.ne' ] ; ) ( show 1 < s by
                                                                           exact one_lt_one_div hs_pos ( by ring_nf at *; nlinarith [ mul_inv_cancel₀ ( ne_of_gt hr ) ] ) ; ) ( show 1 < t by
                                                                                                   rw [ lt_div_iff₀ ] <;> norm_num at *;
                                                                                                   · linarith [ inv_pos.2 ( zero_lt_one.trans hp ), inv_pos.2 ( zero_lt_one.trans hq ), inv_pos.2 hr ];
                                                                                                   · grobner ) _ _ _ _ using 1;
                 · simp +decide only [← rpow_mul];
                 · grind;
                 · fun_prop;
                 · fun_prop;
                 · exact hg.pow_const _ |> Measurable.aemeasurable;
  convert h_holder using 3;
  · simp +decide [ mul_assoc, mul_comm, mul_left_comm, ← ENNReal.rpow_add_of_nonneg, div_nonneg ( zero_le_one.trans hp.le ) hr.le, div_nonneg ( zero_le_one.trans hq.le ) hr.le ];
    simp +decide [ ← mul_assoc, ← ENNReal.rpow_add_of_nonneg, show 0 ≤ p / r by positivity, show 0 ≤ q / r by positivity, show 0 ≤ 1 - p / r by nlinarith [ div_mul_cancel₀ p hr.ne', div_mul_cancel₀ q hr.ne', div_mul_cancel₀ 1 hr.ne', div_mul_cancel₀ 1 ( by positivity : ( p : ℝ ) ≠ 0 ), div_mul_cancel₀ 1 ( by positivity : ( q : ℝ ) ≠ 0 ) ], show 0 ≤ 1 - q / r by nlinarith [ div_mul_cancel₀ p hr.ne', div_mul_cancel₀ q hr.ne', div_mul_cancel₀ 1 hr.ne', div_mul_cancel₀ 1 ( by positivity : ( p : ℝ ) ≠ 0 ), div_mul_cancel₀ 1 ( by positivity : ( q : ℝ ) ≠ 0 ) ] ];
    rw [ mul_right_comm, ← ENNReal.rpow_add_of_nonneg ] <;> norm_num;
    · positivity;
    · ring_nf at *; nlinarith [ inv_pos.2 hr, inv_pos.2 ( zero_lt_one.trans hp ), inv_pos.2 ( zero_lt_one.trans hq ), mul_inv_cancel₀ ( ne_of_gt hr ), mul_inv_cancel₀ ( ne_of_gt ( zero_lt_one.trans hp ) ), mul_inv_cancel₀ ( ne_of_gt ( zero_lt_one.trans hq ) ) ] ;
  · congr! 2;
    ext y; rw [ ENNReal.mul_rpow_of_nonneg _ _ ( by positivity ) ] ; rw [ ← ENNReal.rpow_mul, ← ENNReal.rpow_mul ] ; ring_nf; norm_num [ hr.ne' ] ;
  · simp +zetaDelta at *;
    rw [ show ( 1 - p / r ) * ( p⁻¹ - r⁻¹ ) ⁻¹ = p by rw [ ← div_eq_mul_inv ] ; rw [ div_eq_iff ] <;> nlinarith [ mul_inv_cancel₀ ( by linarith : p ≠ 0 ), mul_inv_cancel₀ ( by linarith : r ≠ 0 ), mul_div_cancel₀ p ( by linarith : r ≠ 0 ) ] ];
    rw [ ← lintegral_comp_sub_left ];
  · rw [ show ( 1 - q / r ) * t = q by rw [ mul_div, div_eq_iff ] <;> nlinarith [ one_div_mul_cancel ( show r ≠ 0 by positivity ), one_div_mul_cancel ( show q ≠ 0 by positivity ), one_div_mul_cancel ( show p ≠ 0 by positivity ), div_mul_cancel₀ q hr.ne' ] ];
  · rw [ one_div_one_div ]

$$

$$
