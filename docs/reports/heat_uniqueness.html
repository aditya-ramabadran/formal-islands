Informal proof:

import Mathlib.MeasureTheory.Integral.IntervalIntegral.IntegrationByParts

open MeasureTheory intervalIntegral Set

/-!
# Energy dissipation for the heat equation

We formalize the energy method calculation showing that the energy
E(t) = (1/2) ∫₀ᴸ w(x,t)² dx is non-increasing when w satisfies
the heat equation w_t = w_xx with homogeneous Dirichlet boundary conditions.

The key steps are:
1. E'(t) = ∫₀ᴸ w · w_t dx  (Leibniz rule / differentiation under the integral)
2. = ∫₀ᴸ w · w_xx dx  (using PDE: w_t = w_xx)
3. = [w · w_x]₀ᴸ - ∫₀ᴸ (w_x)² dx  (integration by parts)
4. = -∫₀ᴸ (w_x)² dx  (boundary conditions: w(0,t) = w(L,t) = 0)
5. ≤ 0  (integrand is nonneg)
-/

/-
Integration by parts, rearranged form: ∫ u * v' = u(b)*v(b) - u(a)*v(a) - ∫ u' * v.
This is a direct consequence of `integral_mul_deriv_eq_deriv_mul`.
-/
lemma ibp_rearranged (a b : ℝ) (u v u' v' : ℝ → ℝ)
    (hu : ∀ x ∈ Set.uIcc a b, HasDerivAt u (u' x) x)
    (hv : ∀ x ∈ Set.uIcc a b, HasDerivAt v (v' x) x)
    (hu' : IntervalIntegrable u' volume a b)
    (hv' : IntervalIntegrable v' volume a b) :
    ∫ x in a..b, u x * v' x =
      u b * v b - u a * v a - ∫ x in a..b, u' x * v x := by
  exact intervalIntegral.integral_mul_deriv_eq_deriv_mul hu hv hu' hv'

/-
The integral of a nonneg function (square) over an interval [0,L] with 0 < L is nonneg.
-/
lemma integral_sq_nonneg (L : ℝ) (hL : 0 < L) (f : ℝ → ℝ) :
    0 ≤ ∫ x in (0 : ℝ)..L, f x * f x := by
  exact intervalIntegral.integral_nonneg hL.le fun x hx => mul_self_nonneg _

Let `w(x,t)` be a sufficiently regular function satisfying the heat equation
`w_t = w_xx` on `(0,L)` with boundary conditions `w(0,t) = w(L,t) = 0`.
Define the energy `E(t) = (1/2) ∫₀ᴸ w(x,t)² dx`. Then `E'(t) ≤ 0`.

The hypothesis `hE_deriv` encodes the Leibniz rule (differentiation under
the integral sign), which is the regularity consequence of `w` being
"sufficiently regular". The remaining proof uses the PDE, integration by
parts, and the boundary conditions to show nonpositivity. -/
theorem energy_dissipation_aristotle
    (L : ℝ) (hL : 0 < L) (t₀ : ℝ)
    -- The function w and its partial derivatives
    (w w_x w_xx : ℝ → ℝ → ℝ)
    -- PDE: w_t = w_xx at time t₀
    (w_t : ℝ → ℝ → ℝ)
    (hPDE : ∀ x, w_t x t₀ = w_xx x t₀)
    -- Boundary conditions at time t₀
    (hbc0 : w 0 t₀ = 0)
    (hbcL : w L t₀ = 0)
    -- Spatial derivatives: w(·, t₀) has derivative w_x(·, t₀)
    (hw_deriv : ∀ x ∈ Set.uIcc 0 L,
      HasDerivAt (fun y => w y t₀) (w_x x t₀) x)
    -- w_x(·, t₀) has derivative w_xx(·, t₀)
    (hwx_deriv : ∀ x ∈ Set.uIcc 0 L,
      HasDerivAt (fun y => w_x y t₀) (w_xx x t₀) x)
    -- Integrability conditions
    (hwx_int : IntervalIntegrable (fun x => w_x x t₀) volume 0 L)
    (hwxx_int : IntervalIntegrable (fun x => w_xx x t₀) volume 0 L)
    -- Energy function and its derivative via the Leibniz rule
    (E : ℝ → ℝ)
    (hE_deriv : HasDerivAt E (∫ x in (0 : ℝ)..L, w x t₀ * w_t x t₀) t₀)
    : deriv E t₀ ≤ 0 := by
  -- Step 1: deriv E t₀ = ∫ w · w_t
  rw [hE_deriv.deriv]
  -- Step 2: Rewrite using PDE: w_t = w_xx
  simp_rw [hPDE]
  -- Step 3: Integration by parts: ∫ w · w_xx = [w·w_x]₀ᴸ - ∫ w_x · w_x
  have h_ibp : ∫ x in (0 : ℝ)..L, w x t₀ * w_xx x t₀ =
      w L t₀ * w_x L t₀ - w 0 t₀ * w_x 0 t₀ -
      ∫ x in (0 : ℝ)..L, w_x x t₀ * w_x x t₀ :=
    ibp_rearranged 0 L (fun y => w y t₀) (fun y => w_x y t₀)
      (fun y => w_x y t₀) (fun y => w_xx y t₀)
      hw_deriv hwx_deriv hwx_int hwxx_int
  rw [h_ibp]
  -- Step 4: Boundary conditions make boundary terms vanish
  simp [hbc0, hbcL]
  -- Step 5: -∫ (w_x)² ≤ 0, i.e., ∫ (w_x)² ≥ 0
  exact integral_sq_nonneg L hL (fun x => w_x x t₀)

Informal statement:

Let \(u,v: \mathbb{R} \times \mathbb{R} \to \mathbb{R}\) be functions such that \(u(x,0) = v(x,0)\) for all \(x\). Let the energy \(E(t) = \frac{1}{2}\int_0^L (u(x,t)-v(x,t))^2\,dx\) be continuous on \([0,T]\), differentiable on \((0,T)\), and satisfy \(E'(t) \le 0\) for \(t \in (0,T)\). Furthermore, suppose \(x \mapsto u(x,t)-v(x,t)\) is continuous on \([0,L]\) and its square is integrable on \([0,L]\) for each \(t \in [0,T]\). Then \(u(x,t) = v(x,t)\) for all \(x \in [0,L]\) and \(t \in [0,T]\).

Informal proof:

This node records a verified supporting sublemma extracted from the formalization of parent node 'heat_uniqueness'. Let \(w(x,t) = u(x,t) - v(x,t)\). Since \(u\) and \(v\) share the same initial data, we have \(w(x,0) = 0\), which implies \(E(0) = \frac{1}{2}\int_0^L 0^2 \, dx = 0\). We are given that \(E\) is continuous on \([0,T]\) and its derivative satisfies \(E'(t) \le 0\) on the interior \((0,T)\). Therefore, \(E(t)\) is non-increasing on \([0,T]\), so for any \(t \in [0,T]\), \(E(t) \le E(0) = 0\). On the other hand, the integrand \(w(x,t)^2\) is non-negative, so \(E(t) \ge 0\). This forces \(E(t) = 0\) for all \(t \in [0,T]\). Because \(w(x,t)\) is continuous with respect to \(x\) on \([0,L]\), the vanishing of the integral of the non-negative continuous function \(w(x,t)^2\) implies that \(w(x,t) = 0\) for all \(x \in [0,L]\). Hence, \(u(x,t) = v(x,t)\) for all \(x \in [0,L]\) and \(t \in [0,T]\).

/-!
# Heat equation uniqueness by the energy method

We formalize the uniqueness theorem for the heat equation using the energy method.
Given two sufficiently regular solutions u, v of the heat equation with the same
initial data and homogeneous Dirichlet boundary conditions, we show u = v.

The proof uses the energy E(t) = (1/2) ∫₀ᴸ (u-v)² dx:
1. E'(t) ≤ 0 (from energy dissipation — a verified child lemma)
2. E(0) = 0 (from equal initial data)
3. E(t) ≥ 0 (integral of squares)
4. Therefore E(t) = 0 for all t ∈ [0,T]
5. Therefore u = v pointwise by continuity
-/

/-- If f is continuous on [0,L] with 0 < L and ∫₀ᴸ f(x)² dx = 0,
    then f(x) = 0 for all x ∈ [0,L]. This is used to go from the
    vanishing energy E(t) = 0 to pointwise equality u = v. -/
lemma eq_zero_of_sq_integral_zero
    (L : ℝ) (hL : 0 < L)
    (f : ℝ → ℝ)
    (hf_cont : ContinuousOn f (Set.Icc 0 L))
    (hf_int : IntervalIntegrable (fun x => f x ^ 2) volume 0 L)
    (hf_zero : ∫ x in (0:ℝ)..L, f x ^ 2 = 0)
    : ∀ x ∈ Set.Icc 0 L, f x = 0 := by
  -- Since f² is nonneg and its integral is zero, f² =ᵐ 0 on [0, L],
  -- hence f =ᵐ 0 on [0, L].
  have h_ae_zero : ∀ᵐ x ∂Measure.restrict volume (Set.Icc 0 L), f x = 0 := by
    rw [integral_of_le hL.le] at hf_zero
    rw [MeasureTheory.integral_eq_zero_iff_of_nonneg (fun x => sq_nonneg _)] at hf_zero
    · simpa [Measure.restrict_congr_set Ioc_ae_eq_Icc] using
        hf_zero.mono fun x hx => sq_eq_zero_iff.mp hx
    · exact hf_int.1
  -- If f(x₀) ≠ 0 for some x₀ ∈ [0,L], continuity gives a neighborhood
  -- where f ≠ 0, contradicting the ae-zero condition.
  intro x hx
  by_contra h_nonzero
  have h_open_interval : ∃ ε > 0, ∀ y ∈ Set.Icc 0 L, |y - x| < ε → f y ≠ 0 := by
    have := Metric.continuousOn_iff.mp hf_cont x hx
    exact Exists.elim (this (|f x|) (abs_pos.mpr h_nonzero)) fun δ hδ =>
      ⟨δ, hδ.1, fun y hy hy' => by
        cases abs_cases (f x) <;> linarith [abs_lt.mp (hδ.2 y hy hy')]⟩
  obtain ⟨ε, ε_pos, hε⟩ := h_open_interval
  rw [ae_restrict_iff'] at * <;> simp_all +decide [Set.Icc_def]
  have h_measure_pos :
      0 < MeasureSpace.volume (Set.Ioo (max 0 (x - ε)) (min L (x + ε))) := by
    simp +zetaDelta at *
    exact ⟨⟨hL, by linarith⟩, by linarith, by linarith⟩
  exact h_measure_pos.ne' (measure_mono_null
    (fun y hy => by
      exact fun h => hε y
        (by cases max_cases 0 (x - ε) <;> linarith [hy.1, hy.2])
        (by cases min_cases L (x + ε) <;> linarith [hy.1, hy.2])
        (abs_lt.mpr ⟨by cases max_cases 0 (x - ε) <;> linarith [hy.1, hy.2],
                      by cases min_cases L (x + ε) <;> linarith [hy.1, hy.2]⟩)
        <| h (by cases max_cases 0 (x - ε) <;> linarith [hy.1, hy.2])
             (by cases min_cases L (x + ε) <;> linarith [hy.1, hy.2]))
    h_ae_zero)

/-- **Heat equation uniqueness by the energy method.**

Let u, v be sufficiently regular solutions of the heat equation on (0,L) × [0,T]
with the same initial data u(x,0) = v(x,0). Define the energy
E(t) = (1/2) ∫₀ᴸ (u(x,t) - v(x,t))² dx. The hypothesis `hE_deriv_nonpos`
encodes the consequence of the energy dissipation lemma (a verified child result):
E'(t) ≤ 0 for t in the interior of [0,T].

The proof shows:
- E is nonincreasing on [0,T] (from E' ≤ 0 via `antitoneOn_of_deriv_nonpos`),
- E(0) = 0 (from equal initial data),
- E(t) ≥ 0 (integral of squares is nonneg),
- therefore E(t) = 0 for all t ∈ [0,T],
- therefore u = v pointwise by continuity of u - v. -/
theorem heat_uniqueness_aristotle
    (L T : ℝ) (hL : 0 < L) (hT : 0 ≤ T)
    (u v : ℝ → ℝ → ℝ)
    -- Same initial data
    (h_init : ∀ x, u x 0 = v x 0)
    -- Energy function E(t) = (1/2) ∫₀ᴸ (u(x,t) - v(x,t))² dx
    (E : ℝ → ℝ)
    (hE_def : ∀ t, E t = (1/2) * ∫ x in (0:ℝ)..L, (u x t - v x t) ^ 2)
    -- E is continuous on [0,T] and differentiable on interior (regularity)
    (hE_cont : ContinuousOn E (Set.Icc 0 T))
    (hE_diff : DifferentiableOn ℝ E (interior (Set.Icc 0 T)))
    -- E' ≤ 0 on interior of [0,T] (from energy dissipation lemma)
    (hE_deriv_nonpos : ∀ t ∈ interior (Set.Icc 0 T), deriv E t ≤ 0)
    -- Continuity of u(·,t) - v(·,t) on [0,L] at each time
    (hw_cont : ∀ t ∈ Set.Icc 0 T,
      ContinuousOn (fun x => u x t - v x t) (Set.Icc 0 L))
    -- Integrability of the squared difference at each time
    (hw_sq_int : ∀ t ∈ Set.Icc 0 T,
      IntervalIntegrable (fun x => (u x t - v x t) ^ 2) volume 0 L)
    : ∀ x t, t ∈ Set.Icc 0 T → x ∈ Set.Icc 0 L → u x t = v x t := by
  -- Step 1: Show E(t) = 0 for all t ∈ [0, T]
  have hE_zero : ∀ t ∈ Set.Icc 0 T, E t = 0 := by
    -- E is nonincreasing on [0, T] (from E' ≤ 0 and continuity)
    have hE_noninc : AntitoneOn E (Set.Icc 0 T) :=
      antitoneOn_of_deriv_nonpos (convex_Icc _ _) hE_cont hE_diff hE_deriv_nonpos
    -- E(0) = 0 (from equal initial data: u x 0 - v x 0 = 0)
    have hE_zero_initial : E 0 = 0 := by aesop
    -- For t ∈ [0,T]: E(t) ≤ E(0) = 0 (antitone) and E(t) ≥ 0 (integral of squares)
    exact fun t ht => le_antisymm
      (hE_noninc (left_mem_Icc.mpr hT) ht ht.1 |> le_trans <| by linarith)
      (hE_def t ▸ mul_nonneg (by norm_num)
        (integral_nonneg (by linarith [ht.1]) fun x hx => sq_nonneg _))
  -- Step 2: From E(t) = 0, deduce u = v pointwise
  intro x t ht hx
  -- E(t) = 0 means (1/2) * ∫ (u-v)² = 0, hence ∫ (u-v)² = 0
  have hE_t := hE_zero t ht
  rw [hE_def] at hE_t
  have hint_zero : ∫ y in (0:ℝ)..L, (u y t - v y t) ^ 2 = 0 := by linarith [hE_t]
  -- By the helper lemma, u(x,t) - v(x,t) = 0 for all x ∈ [0,L]
  have := eq_zero_of_sq_integral_zero L hL (fun y => u y t - v y t)
    (hw_cont t ht) (hw_sq_int t ht) hint_zero x hx
  linarith
