Two-point logarithmic Sobolev inequality

Two-point logarithmic Sobolev inequality. Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), define the entropy of \(f^2\) by \[ \operatorname{Ent}_{\mu}(f^2) := \mathbb{E}_{\mu}[f^2 \log f^2] - \mathbb{E}_{\mu}[f^2] \log \bigl(\mathbb{E}_{\mu}[f^2]\bigr), \] with the convention \(0\log 0 := 0\). Then \[ \operatorname{Ent}_{\mu}(f^2) \le \frac{1}{2}\bigl(f(1)-f(-1)\bigr)^2. \] Equivalently, if \(a=f(1)\) and \(b=f(-1)\), then \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \] This is the logarithmic Sobolev inequality on the two-point Bernoulli space, and it is the basic finite-state model underlying tensorization arguments toward the Gaussian logarithmic Sobolev inequality.

Informal statement:

For every \(f:\{-1,1\}\to\mathbb{R}\), \(\operatorname{Ent}_{\mu}(f^2)\le\tfrac{1}{2}(f(1)-f(-1))^2\), where \(\mu\) is the uniform measure on \(\{-1,1\}\).

Informal proof:

Set \(a:=f(1)\) and \(b:=f(-1)\). Since \(\mu\) is uniform, \(\mathbb{E}_\mu[f^2]=(a^2+b^2)/2\) and \(\mathbb{E}_\mu[f^2\log f^2]=(a^2\log(a^2)+b^2\log(b^2))/2\). That scalar inequality is established in the node below.

Display label: Scalar inequality

Formalization priority: 2. Rationale: The sign reduction (using (|a|-|b|)^2 <= (a-b)^2) and the homogeneity normalization are nontrivial algebraic steps. The cancellation of log(c^2) cross-terms in the scaling argument merits formal verification. Lower priority than the core 1D node because these steps are more routine once the framework is in place.

Informal statement:

For all \(a,b\in\mathbb{R}\), \[a^2\log(a^2)+b^2\log(b^2)-(a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right)\le(a-b)^2.\]

Informal proof:

Step 1 (sign reduction). The left-hand side depends only on \(a^2\) and \(b^2\), so replacing \((a,b)\) by \((|a|,|b|)\) leaves the LHS unchanged. The reverse triangle inequality gives \((|a|-|b|)^2\le(a-b)^2\), so it suffices to prove the inequality for \(a,b\ge 0\). Step 2 (homogeneity normalization). Replacing \((a,b)\) by \((ca,cb)\) multiplies both sides by \(c^2\) (the \(\log(c^2)\) cross-terms cancel on the left). Hence the inequality is homogeneous of degree 2. If \(a=b=0\) the claim is trivial; otherwise scale by \(c=\sqrt{2/(a^2+b^2)}\) to reduce to \(a^2+b^2=2\). Under this normalization the desired inequality becomes \(a^2\log(a^2)+b^2\log(b^2)\le(a-b)^2\). Step 3 (one-variable substitution). With \(a,b\ge0\) and \(a^2+b^2=2\), set \(u=a^2\in[0,2]\) so \(b^2=2-u\), \(a=\sqrt{u}\), \(b=\sqrt{2-u}\). The inequality becomes \(u\log u+(2-u)\log(2-u)\le(\sqrt{u}-\sqrt{2-u})^2\), i.e., the core one-dimensional inequality \(G(u)\ge0\) established in the next node.
