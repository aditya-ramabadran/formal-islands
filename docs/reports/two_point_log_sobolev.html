Two-point logarithmic Sobolev inequality

Two-point logarithmic Sobolev inequality. Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), define the entropy of \(f^2\) by \[ \operatorname{Ent}_{\mu}(f^2) := \mathbb{E}_{\mu}[f^2 \log f^2] - \mathbb{E}_{\mu}[f^2] \log \bigl(\mathbb{E}_{\mu}[f^2]\bigr), \] with the convention \(0\log 0 := 0\). Informal statement:

Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), we have \[ \operatorname{Ent}_{\mu}(f^2) \le \frac{1}{2}\bigl(f(1)-f(-1)\bigr)^2. \]

Informal proof:

Write \[ a := f(1), \qquad b := f(-1). \] Because \(\mu\) is the uniform measure on two points, \[ \mathbb{E}_{\mu}[f^2] = \frac{a^2+b^2}{2} \] and \[ \mathbb{E}_{\mu}[f^2\log f^2] = \frac{a^2\log(a^2)+b^2\log(b^2)}{2}. \] Therefore the entropy inequality \[ \operatorname{Ent}_{\mu}(f^2) \le \frac12 (a-b)^2 \] is exactly equivalent, after multiplying both sides by \(2\), to \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \] By the scalar inequality lemma, this holds for all real \(a,b\). Dividing by \(2\) yields the claimed logarithmic Sobolev inequality.

We prove the logarithmic Sobolev inequality on the two-point Bernoulli space {-1, 1}

  Ent_μ(f²) ≤ (1/2) · (f(1) - f(-1))²

/-- Expectation under the uniform measure on {-1, 1}.

def twoPointExpect (g : TwoPoint → ℝ) : ℝ :=
  (g false + g true) / 2

  twoPointExpect (fun x => g x * Real.log (g x)) -
  twoPointExpect g * Real.log (twoPointExpect g)

**Normalized scalar inequality (Two-point logarithmic Sobolev).**
For every `u ∈ [0, 2]`,
  `u * log u + (2 - u) * log (2 - u) ≤ (√u - √(2 - u))²`,
with the convention `0 * log 0 = 0` (which holds in Mathlib since `Real.log 0 = 0`).

**Proof strategy:**
We reduce to showing `(1+t) log(1+t) + (1-t) log(1-t) ≤ 2 - 2√(1-t²)` for `t ∈ [0,1]`
via substitution `t = u - 1`. The key chain of lemmas is:

1. `√(1+x²) ≤ 1 + x²/2`  (squaring both sides)
2. `x + √(1+x²) ≤ exp x`  for `x ≥ 0`  (from exp ≥ 1+x+x²/2 and step 1)
3. `log(x + √(1+x²)) ≤ x`  for `x ≥ 0`  (log of step 2)
4. `log((1+t)/(1-t)) ≤ 2t/√(1-t²)`  for `t ∈ [0,1)`  (substitution in step 3)
5. `(1+t) log(1+t) + (1-t) log(1-t) ≤ 2 - 2√(1-t²)` (MVT + step 4)
-/

/-
√(1 + x²) ≤ 1 + x²/2 for all real x.
-/
lemma sqrt_one_add_sq_le (x : ℝ) : sqrt (1 + x ^ 2) ≤ 1 + x ^ 2 / 2 := by
  exact Real.sqrt_le_iff.mpr ⟨ by positivity, by nlinarith ⟩

/-
For x ≥ 0, x + √(1+x²) ≤ exp x.
Proof: exp x ≥ 1 + x + x²/2 and √(1+x²) ≤ 1 + x²/2, so
x + √(1+x²) ≤ x + 1 + x²/2 ≤ exp x.
-/
lemma add_sqrt_le_exp (x : ℝ) (hx : 0 ≤ x) : x + sqrt (1 + x ^ 2) ≤ exp x := by
  -- Use the fact that $e^x \geq 1 + x + x^2 / 2$ for all $x \geq 0$.
  have exp_bound : Real.exp x ≥ 1 + x + x ^ 2 / 2 := by
    exact quadratic_le_exp_of_nonneg hx;
  nlinarith [ Real.mul_self_sqrt ( by positivity : 0 ≤ 1 + x ^ 2 ) ]

/-
For x ≥ 0, log(x + √(1+x²)) ≤ x.
Follows from add_sqrt_le_exp by taking log of both sides.
-/
lemma log_add_sqrt_le (x : ℝ) (hx : 0 ≤ x) : log (x + sqrt (1 + x ^ 2)) ≤ x := by
  exact Real.log_le_iff_le_exp ( by positivity ) |>.2 ( add_sqrt_le_exp x hx )

/-
For 0 ≤ t < 1: log((1+t)/(1-t)) ≤ 2t/√(1-t²).
Proof: Set x = t/√(1-t²) ≥ 0. Then √(1+x²) = 1/√(1-t²), and
x + √(1+x²) = (t+1)/√(1-t²) = √((1+t)/(1-t)).
So log(x+√(1+x²)) = ½ log((1+t)/(1-t)) ≤ x = t/√(1-t²) by log_add_sqrt_le.
Multiply by 2.
-/
lemma log_div_le (t : ℝ) (ht0 : 0 ≤ t) (ht1 : t < 1) :
    log ((1 + t) / (1 - t)) ≤ 2 * t / sqrt (1 - t ^ 2) := by
  -- Set x = t / √(1-t²). Since 0 ≤ t < 1, we have 1 - t² > 0, so √(1-t²) > 0 and x ≥ 0.
  set x := t / Real.sqrt (1 - t ^ 2) with hx;
  -- Then √(1+x²) = 1/√(1-t²), and x + √(1+x²) = (t+1)/√(1-t²) = √((1+t)/(1-t)).
  have h_sqrt : Real.sqrt (1 + x ^ 2) = 1 / Real.sqrt (1 - t ^ 2) := by
    rw [ hx, div_pow, Real.sq_sqrt <| by nlinarith ];
    rw [ one_add_div ] <;> norm_num ; nlinarith
  have h_expr : x + Real.sqrt (1 + x ^ 2) = Real.sqrt ((1 + t) / (1 - t)) := by
    rw [ h_sqrt, show ( 1 + t ) / ( 1 - t ) = ( ( 1 + t ) / Real.sqrt ( 1 - t ^ 2 ) ) ^ 2 by rw [ div_pow, Real.sq_sqrt <| by nlinarith ] ; rw [ div_eq_div_iff ] <;> nlinarith, Real.sqrt_sq <| by exact div_nonneg ( by linarith ) <| Real.sqrt_nonneg _ ] ; ring;
  -- So log(x+√(1+x²)) = ½ log((1+t)/(1-t)) ≤ x = t/√(1-t²) by log_add_sqrt_le.
  have h_log : Real.log (x + Real.sqrt (1 + x ^ 2)) ≤ x := by
    exact log_add_sqrt_le x ( by positivity );
  rw [ h_expr, Real.log_sqrt ( div_nonneg ( by linarith ) ( by linarith ) ) ] at h_log ; ring_nf at * ; linarith

/-
For 0 ≤ t ≤ 1: (1+t) log(1+t) + (1-t) log(1-t) ≤ 2 - 2√(1-t²).
This is proved by showing the function
  h(t) = 2 - 2√(1-t²) - (1+t)log(1+t) - (1-t)log(1-t)
satisfies h(0) = 0 and h'(t) = 2t/√(1-t²) - log((1+t)/(1-t)) ≥ 0 for t ∈ (0,1),
hence h is nondecreasing on [0,1). At t=1 separately: 2 log 2 ≤ 2.
-/
lemma entropy_le (t : ℝ) (ht0 : 0 ≤ t) (ht1 : t ≤ 1) :
    (1 + t) * log (1 + t) + (1 - t) * log (1 - t) ≤ 2 - 2 * sqrt (1 - t ^ 2) := by
  -- We'll use the fact that $h(t)$ is differentiable and apply the mean value theorem.
  have h_deriv_nonneg : ∀ t ∈ Set.Ioo 0 1, 0 ≤ deriv (fun t => 2 - 2 * Real.sqrt (1 - t^2) - (1 + t) * Real.log (1 + t) - (1 - t) * Real.log (1 - t)) t := by
    -- Let's calculate the derivative of $h(t)$ using the sum and chain rules.
    have h_deriv : ∀ t ∈ Set.Ioo 0 1, deriv (fun t => 2 - 2 * Real.sqrt (1 - t^2) - (1 + t) * Real.log (1 + t) - (1 - t) * Real.log (1 - t)) t = 2 * t / Real.sqrt (1 - t^2) - Real.log ((1 + t) / (1 - t)) := by
      intro t ht; refine' HasDerivAt.deriv _;
      convert HasDerivAt.sub ( HasDerivAt.sub ( HasDerivAt.sub ( hasDerivAt_const _ _ ) ( HasDerivAt.mul ( hasDerivAt_const _ _ ) ( HasDerivAt.sqrt ( HasDerivAt.sub ( hasDerivAt_const _ _ ) ( hasDerivAt_pow 2 t ) ) _ ) ) ) ( HasDerivAt.mul ( hasDerivAt_id' t |> HasDerivAt.const_add _ ) ( HasDerivAt.log ( hasDerivAt_id' t |> HasDerivAt.const_add _ ) _ ) ) ) ( HasDerivAt.mul ( hasDerivAt_id' t |> HasDerivAt.const_sub _ ) ( HasDerivAt.log ( hasDerivAt_id' t |> HasDerivAt.const_sub _ ) _ ) ) using 1 <;> norm_num;
      · rw [ Real.log_div ] <;> ring <;> nlinarith [ ht.1, ht.2, mul_inv_cancel₀ ( by linarith [ ht.1, ht.2 ] : ( 1 + t ) ≠ 0 ), mul_inv_cancel₀ ( by linarith [ ht.1, ht.2 ] : ( 1 - t ) ≠ 0 ) ];
      · nlinarith [ ht.1, ht.2 ];
      · linarith [ ht.1, ht.2 ];
      · linarith [ ht.1, ht.2 ];
    -- Apply the lemma log_div_le to conclude that the derivative is non-negative.
    intros t ht
    rw [h_deriv t ht]
    apply sub_nonneg_of_le
    apply log_div_le t ht.1.le ht.2;
  by_contra h_contra;
  -- Apply the mean value theorem to the interval $[0, t]$.
  obtain ⟨c, hc⟩ : ∃ c ∈ Set.Ioo 0 t, deriv (fun t => 2 - 2 * Real.sqrt (1 - t^2) - (1 + t) * Real.log (1 + t) - (1 - t) * Real.log (1 - t)) c = (2 - 2 * Real.sqrt (1 - t^2) - (1 + t) * Real.log (1 + t) - (1 - t) * Real.log (1 - t) - (2 - 2 * Real.sqrt (1 - 0^2) - (1 + 0) * Real.log (1 + 0) - (1 - 0) * Real.log (1 - 0))) / (t - 0) := by
    apply_rules [ exists_deriv_eq_slope ];
    · exact ht0.lt_of_ne ( by rintro rfl; norm_num at h_contra );
    · fun_prop;
    · exact fun x hx => DifferentiableAt.differentiableWithinAt ( by exact DifferentiableAt.sub ( DifferentiableAt.sub ( DifferentiableAt.sub ( differentiableAt_const _ ) ( DifferentiableAt.mul ( differentiableAt_const _ ) ( DifferentiableAt.sqrt ( by norm_num ) ( by nlinarith [ hx.1, hx.2 ] ) ) ) ) ( DifferentiableAt.mul ( differentiableAt_id.const_add _ ) ( DifferentiableAt.log ( differentiableAt_id.const_add _ ) ( by linarith [ hx.1, hx.2 ] ) ) ) ) ( DifferentiableAt.mul ( differentiableAt_id.const_sub _ ) ( DifferentiableAt.log ( differentiableAt_id.const_sub _ ) ( by linarith [ hx.1, hx.2 ] ) ) ) );
  have := h_deriv_nonneg c ⟨ hc.1.1, hc.1.2.trans_le ht1 ⟩ ; rw [ hc.2, le_div_iff₀ ] at this <;> norm_num at * <;> linarith;

/-
The main normalized inequality: for u ∈ [0, 2],
  u log u + (2-u) log(2-u) ≤ (√u - √(2-u))².
Proof: substitute t = |u - 1| so t ∈ [0, 1], and apply entropy_le.
-/
theorem normalized_inequality_aristotle (u : ℝ) (hu0 : 0 ≤ u) (hu2 : u ≤ 2) :
    u * Real.log u + (2 - u) * Real.log (2 - u) ≤
    (Real.sqrt u - Real.sqrt (2 - u)) ^ 2 := by
  -- Substitute $t = |u - 1|$ so $t \in [0, 1]$, and apply entropy_le.
  have h_subst : ∃ t ∈ Set.Icc 0 1, u = 1 + t ∨ u = 1 - t := by
    exact ⟨ |u - 1|, ⟨ abs_nonneg _, abs_le.mpr ⟨ by linarith, by linarith ⟩ ⟩, by cases abs_cases ( u - 1 ) <;> first | left; linarith | right; linarith ⟩;
  -- For the RHS, expand (√u - √(2-u))² = u + (2-u) - 2√(u(2-u)) = 2 - 2√(u(2-u)) and show u(2-u) = 1 - (u-1)² = 1 - t².
  have h_rhs : (Real.sqrt u - Real.sqrt (2 - u)) ^ 2 = 2 - 2 * Real.sqrt (u * (2 - u)) := by
    rw [ Real.sqrt_mul ] <;> nlinarith [ Real.mul_self_sqrt hu0, Real.mul_self_sqrt ( by linarith : 0 ≤ 2 - u ) ];
  rcases h_subst with ⟨ t, ⟨ ht0, ht1 ⟩, rfl | rfl ⟩ <;> ( have := entropy_le t ht0 ht1; ring_nf at * ; aesop; )
