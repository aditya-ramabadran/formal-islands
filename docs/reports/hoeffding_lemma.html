Hoeffding's lemma

Hoeffding's lemma. Let \(X\) be a real-valued random variable such that \[ \mathbb{E}[X]=0 \] and suppose there exist real numbers \(a<b\) such that \[ a \le X \le b \qquad\text{almost surely}. \] Then for every real \(s\), \[ \mathbb{E}\bigl[e^{sX}\bigr] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right). \] This is the classical Hoeffding moment-generating-function bound for a centered bounded random variable.

Node id: hoeffdings_lemma | Status: informal

Used by (parent nodes): convexity_expectation, log_mgf_bound

Informal statement:

Let \(X\) be a real-valued random variable such that \(\mathbb{E}[X]=0\) and suppose there exist real numbers \(a<b\) such that \(a \le X \le b\) almost surely. Then for every real \(s\), \(\mathbb{E}\bigl[e^{sX}\bigr] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right).\)

Informal proof:

Fix a real number \(s\). Let \(p := \frac{-a}{b-a}\), \(q := \frac{b}{b-a}\), and \(u := s(b-a)\). Since \(a \le X \le b\) and \(\mathbb{E}[X]=0\), we have \(a \le 0 \le b\), which implies \(p, q \ge 0\). Furthermore, \(p+q = \frac{-a+b}{b-a}=1\). By bounding the expectation using convexity, we find that \[ \mathbb{E}[e^{sX}] \le e^{-pu}(q+p e^u). \] We can rewrite the right side as \(e^{g(u)}\), where \(g(u) := -pu + \log(q+p e^u)\). By the log-MGF bound for such functions, we know \(g(u) \le \frac{u^2}{8}\). Therefore, \[ \mathbb{E}[e^{sX}] \le e^{u^2/8}. \] Finally, substitute \(u=s(b-a)\). Then \[ e^{u^2/8} = \exp\!\left(\frac{s^2(b-a)^2}{8}\right). \] Hence \[ \mathbb{E}[e^{sX}] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right), \] which proves Hoeffding's lemma.

Expectation bound via convexity

Node id: convexity_expectation | Status: formal_verified

Depends on (child nodes): hoeffdings_lemma

Formalization priority: 2. Rationale: An excellent intermediate probabilistic step that isolates the use of convexity and linearity of expectation to bound the MGF, separating it from the final algebraic/calculus arguments.

Informal statement:

Let \(X\) be a real-valued random variable with \(\mathbb{E}[X]=0\) and \(a \le X \le b\) almost surely. Let \(s\) be a real number, \(p := \frac{-a}{b-a}\), \(q := \frac{b}{b-a}\), and \(u := s(b-a)\). Then \(\mathbb{E}[e^{sX}] \le e^{-pu}(q+p e^u)\).

Informal proof:

Let \(x \in [a,b]\). Since \(x\) lies between \(a\) and \(b\), we may write it as the convex combination \[ x = \frac{b-x}{b-a}\,a + \frac{x-a}{b-a}\,b. \] By convexity of the function \(t \mapsto e^{st}\), it follows that \[ e^{sx} \le \frac{b-x}{b-a} e^{sa} + \frac{x-a}{b-a} e^{sb}. \] Applying this with \(x=X\), we get almost surely \[ e^{sX} \le \frac{b-X}{b-a} e^{sa} + \frac{X-a}{b-a} e^{sb}. \] Taking expectations and using linearity of expectation gives \[ \mathbb{E}[e^{sX}] \le \frac{b-\mathbb{E}[X]}{b-a} e^{sa} + \frac{\mathbb{E}[X]-a}{b-a} e^{sb}. \] Since \(\mathbb{E}[X]=0\), this simplifies to \[ \mathbb{E}[e^{sX}] \le \frac{b}{b-a} e^{sa} + \frac{-a}{b-a} e^{sb} = q e^{sa} + p e^{sb}. \] Since \(b = a + (b-a)\), we have \[ e^{sb} = e^{s(a+(b-a))} = e^{sa} e^u. \] Hence \[ q e^{sa} + p e^{sb} = q e^{sa} + p e^{sa} e^u = e^{sa}(q + p e^u). \] Also, from the definition of \(p\), we have \(a = -p(b-a)\), so \(sa = -pu\). Thus \[ e^{sa}(q+p e^u) = e^{-pu}(q+p e^u). \] We conclude that \(\mathbb{E}[e^{sX}] \le e^{-pu}(q+p e^u)\).

Log-MGF function bound

Node id: log_mgf_bound | Status: formal_verified

Depends on (child nodes): hoeffdings_lemma

Formalization priority: 1. Rationale: A completely self-contained, purely real-analytic bound. It packages the Taylor expansion and derivative calculations into an elegant, isolated calculus problem, independent of probability.

Informal statement:

Let \(p, q \ge 0\) such that \(p+q=1\). Define \(g(u) := -pu + \log(q+p e^u)\). Then \(g(u) \le \frac{u^2}{8}\) for all \(u \in \mathbb{R}\).

Informal proof:

We analyze \(g\). First, \[ g(0) = -p\cdot 0 + \log(q+p) = \log(1)=0. \] Differentiate: \[ g'(u) = -p + \frac{p e^u}{q+p e^u}. \] In particular, \[ g'(0) = -p + \frac{p}{q+p} = -p+p=0. \] Differentiate again: \[ g''(u) = \frac{p e^u(q+p e^u) - p e^u(p e^u)}{(q+p e^u)^2} = \frac{p q e^u}{(q+p e^u)^2}. \] Set \(y := \frac{p e^u}{q+p e^u}\). Then \(0 \le y \le 1\), and since \[ 1-y = \frac{q}{q+p e^u}, \] we have \[ y(1-y) = \frac{p e^u}{q+p e^u} \cdot \frac{q}{q+p e^u} = \frac{p q e^u}{(q+p e^u)^2} = g''(u). \] But for every \(y \in [0,1]\), \[ y(1-y) \le \frac14. \] Therefore \(g''(u) \le \frac14\) for all \(u\). Now apply Taylor's theorem with remainder in the mean-value form at the point \(0\). For each \(u\), there exists some \(\xi\) between \(0\) and \(u\) such that \[ g(u) = g(0) + u g'(0) + \frac{u^2}{2} g''(\xi). \] Since \(g(0)=0\), \(g'(0)=0\), and \(g''(\xi) \le 1/4\), we obtain \[ g(u) \le \frac{u^2}{2} \cdot \frac14 = \frac{u^2}{8}. \]

  Let p, q ≥ 0 such that p + q = 1. Define g(u) := -pu + log(q + p·eᵘ).
  Then g(u) ≤ u²/8 for all u ∈ ℝ.

  Proof strategy: define h(u) = u²/8 + pu - log(q + p·eᵘ) = u²/8 - g(u).
  Show h is convex (second derivative ≥ 0), h(0) = 0, h'(0) = 0.

private lemma denom_pos {p q : ℝ} (hp : 0 ≤ p) (hq : 0 ≤ q) (hpq : p + q = 1)

  rcases eq_or_lt_of_le hq with rfl | hq'
  · simp at hpq; simp [hpq, exp_pos]
  · have : 0 ≤ p * exp u := mul_nonneg hp (exp_pos u).le
    linarith
