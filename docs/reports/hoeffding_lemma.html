Hoeffding's lemma

Hoeffding's lemma. Let \(X\) be a real-valued random variable such that \[ \mathbb{E}[X]=0 \] and suppose there exist real numbers \(a<b\) such that \[ a \le X \le b \qquad\text{almost surely}. \] Then for every real \(s\), \[ \mathbb{E}\bigl[e^{sX}\bigr] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right). \] This is the classical Hoeffding moment-generating-function bound for a centered bounded random variable.

Informal statement:

Let \(X\) be a real-valued random variable such that \(\mathbb{E}[X]=0\) and suppose there exist real numbers \(a<b\) such that \(a \le X \le b\) almost surely. Then for every real \(s\), \(\mathbb{E}[e^{sX}] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right)\).

Informal proof:

Fix a real number \(s\). Because \(a \le X \le b\) almost surely and the exponential function is convex, we first establish a pointwise bound for \(e^{sx}\) on the interval \([a,b]\). Let \(x \in [a,b]\). Since \(x\) lies between \(a\) and \(b\), we may write it as the convex combination \[ x = \frac{b-x}{b-a}\,a + \frac{x-a}{b-a}\,b, \] where the coefficients are nonnegative and sum to \(1\). By convexity of the function \(t \mapsto e^{st}\), it follows that \[ e^{sx} \le \frac{b-x}{b-a} e^{sa} + \frac{x-a}{b-a} e^{sb}. \] Applying this with \(x=X\), we get almost surely \[ e^{sX} \le \frac{b-X}{b-a} e^{sa} + \frac{X-a}{b-a} e^{sb}. \] Taking expectations and using linearity of expectation gives \[ \mathbb{E}[e^{sX}] \le \frac{e^{sa}}{b-a}\,\mathbb{E}[b-X] + \frac{e^{sb}}{b-a}\,\mathbb{E}[X-a]. \] Now use \(\mathbb{E}[X]=0\). Then \[ \mathbb{E}[b-X] = b-\mathbb{E}[X] = b, \qquad \mathbb{E}[X-a] = \mathbb{E}[X]-a = -a. \] Therefore \[ \mathbb{E}[e^{sX}] \le \frac{b}{b-a} e^{sa} + \frac{-a}{b-a} e^{sb}. \] Set \[ p := \frac{-a}{b-a}, \qquad q := \frac{b}{b-a}. \] Then \(p,q \ge 0\) and \(p+q = \frac{-a+b}{b-a}=1\). So the previous inequality becomes \[ \mathbb{E}[e^{sX}] \le q e^{sa} + p e^{sb}. \] Now let \(u := s(b-a)\). Since \(b = a + (b-a)\), we have \[ e^{sb} = e^{s(a+(b-a))} = e^{sa} e^u. \] Hence \[ q e^{sa} + p e^{sb} = q e^{sa} + p e^{sa} e^u = e^{sa}(q + p e^u). \] Also, from the definition of \(p\), we have \(a = -p(b-a)\), so \(sa = -pu\). Thus \[ e^{sa}(q+p e^u) = e^{-pu}(q+p e^u). \] We conclude that \[ \mathbb{E}[e^{sX}] \le e^{-pu}(q+p e^u). \] By the calculus lemma, for \(g(u) := -pu + \log(q+p e^u)\), we have \(g(u) \le \frac{u^2}{8}\). Exponentiating both sides yields \[ e^{-pu}(q+p e^u) = e^{g(u)} \le e^{u^2/8}. \] Therefore \[ \mathbb{E}[e^{sX}] \le e^{u^2/8}. \] Finally substitute \(u=s(b-a)\). Then \[ e^{u^2/8} = \exp\!\left(\frac{s^2(b-a)^2}{8}\right). \] Hence \[ \mathbb{E}[e^{sX}] \le \exp\!\left(\frac{s^2(b-a)^2}{8}\right). \] This proves Hoeffding's lemma.

Informal statement:

Let \(p, q \ge 0\) be real numbers such that \(p+q=1\). Define \[ g(u) := -pu + \log(q+p e^u). \] Then for all \(u \in \mathbb{R}\), \[ g(u) \le \frac{u^2}{8}. \]

Informal proof:

We now analyze \(g\). First, \[ g(0) = -p\cdot 0 + \log(q+p) = \log(1)=0. \] Differentiate: \[ g'(u) = -p + \frac{p e^u}{q+p e^u}. \] In particular, \[ g'(0) = -p + \frac{p}{q+p} = -p+p=0. \] Differentiate again: \[ g''(u) = \frac{p e^u(q+p e^u) - p e^u(p e^u)}{(q+p e^u)^2} = \frac{p q e^u}{(q+p e^u)^2}. \] This quantity is nonnegative. We now show it is at most \(1/4\). Set \[ y := \frac{p e^u}{q+p e^u}. \] Then \(0 \le y \le 1\), and since \[ 1-y = \frac{q}{q+p e^u}, \] we have \[ y(1-y) = \frac{p e^u}{q+p e^u} \cdot \frac{q}{q+p e^u} = \frac{p q e^u}{(q+p e^u)^2} = g''(u). \] But for every \(y \in [0,1]\), \[ y(1-y) \le \frac14. \] Therefore \[ g''(u) \le \frac14 \qquad\text{for all } u. \] Now apply Taylor's theorem with remainder in the mean-value form at the point \(0\). For each \(u\), there exists some \(\xi\) between \(0\) and \(u\) such that \[ g(u) = g(0) + u g'(0) + \frac{u^2}{2} g''(\xi). \] Since \(g(0)=0\), \(g'(0)=0\), and \(g''(\xi) \le 1/4\), we obtain \[ g(u) \le \frac{u^2}{2} \cdot \frac14 = \frac{u^2}{8}. \]

/-- **Hoeffding's calculus lemma.** Let p, q ≥ 0 with p + q = 1. Then for all u,
    -p·u + log(q + p·eᵘ) ≤ u²/8. -/
theorem hoeffding_calculus_lemma_aristotle (p q : ℝ) (hp : 0 ≤ p) (hq : 0 ≤ q)
    (hpq : p + q = 1) (u : ℝ) :
    -p * u + Real.log (q + p * Real.exp u) ≤ u ^ 2 / 8 := by
  have hq_eq : q = 1 - p := by linarith
  have hp1 : p ≤ 1 := by linarith
  subst hq_eq
  show hoeffding_g p u ≤ u ^ 2 / 8
  rcases le_or_gt 0 u with hu | hu
  · exact hoeffding_bound_pos p u hp hp1 hu
  · exact hoeffding_bound_neg p u hp hp1 hu.le
