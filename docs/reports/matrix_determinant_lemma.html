Matrix determinant lemma

Matrix determinant lemma. Let \(A\) be an invertible \(n\times n\) matrix over \(\mathbb{R}\) (or over any commutative field), and let \(u,v\in \mathbb{R}^n\) be column vectors. Then \[ \det(A+uv^{T})=\det(A)\bigl(1+v^{T}A^{-1}u\bigr). \] Equivalently, if one first proves the special case \[ \det(I+uv^{T})=1+v^{T}u, \] then the general statement follows immediately by factoring \[ A+uv^{T}=A\bigl(I+A^{-1}uv^{T}\bigr)=A\bigl(I+(A^{-1}u)v^{T}\bigr). \] This is the classical matrix determinant lemma for a rank-one update. It is a concrete finite-dimensional algebraic identity that frequently appears inside larger arguments in matrix analysis, numerical linear algebra, random matrix theory, optimization, and perturbation theory.

Formalization priority: 1. Rationale: This is the essential finite-dimensional algebraic core. The multilinearity expansion, vanishing of repeated-u terms, and coordinate evaluation of det(...,u,...) are all concrete, non-trivial steps that Lean must verify explicitly. Once certified, the full lemma reduces to one short algebraic manipulation. Highest-yield formal island in the proof.

Informal statement:

For column vectors \(u,v\in\mathbb{R}^n\), \[\det(I+uv^{T})=1+v^{T}u.\]

Informal proof:

Write the \(j\)-th column of \(I+uv^{T}\) as \(e_j+v_j u\). By multilinearity of the determinant in the columns, \[\det(I+uv^{T})=\det(e_1+v_1 u,\dots,e_n+v_n u)\] expands into a sum over choices of either the \(e_j\) part or the \(v_j u\) part from each column. Any term in which two or more columns contribute the \(u\) part has at least two columns proportional to \(u\), hence is linearly dependent and its determinant vanishes. The surviving terms are: (1) all columns contribute their \(e_j\) part, giving \(\det(e_1,\dots,e_n)=1\); and (2) exactly the \(j\)-th column contributes the \(u\) part, giving \(v_j\det(e_1,\dots,e_{j-1},u,e_{j+1},\dots,e_n)=v_j u_j\), since replacing the \(j\)-th column of the identity matrix by \(u\) yields determinant \(u_j\). Summing over \(j\), \[\det(I+uv^{T})=1+\sum_{j=1}^{n}v_j u_j=1+v^{T}u.\]

Given the special case, its proof is a short chain (factoring, multiplicativity of det, substitution), but formalizing the invertibility-based factorization and applying multiplicativity in Lean carries real but manageable overhead.

Informal statement:

Let \(A\) be an invertible \(n\times n\) matrix over \(\mathbb{R}\) and let \(u,v\in\mathbb{R}^n\). Then \[\det(A+uv^{T})=\det(A)\bigl(1+v^{T}A^{-1}u\bigr).\]

Informal proof:

Since \(A\) is invertible, factor \[A+uv^{T}=A\bigl(I+A^{-1}u\,v^{T}\bigr).\] Set \(w:=A^{-1}u\). By multiplicativity of the determinant, \[\det(A+uv^{T})=\det(A)\det(I+w v^{T}).\] Applying the rank-one identity (special_case) with \(u\mapsto w\), \[\det(I+w v^{T})=1+v^{T}w=1+v^{T}A^{-1}u.\] Hence \(\det(A+uv^{T})=\det(A)\bigl(1+v^{T}A^{-1}u\bigr)\).

import Mathlib

/-!
# Matrix Determinant Lemma

The **matrix determinant lemma** states that for an invertible n×n matrix `A` over ℝ
and column vectors `u, v ∈ ℝⁿ`:

  det(A + u vᵀ) = det(A) · (1 + vᵀ A⁻¹ u)

The outer product `u vᵀ` is expressed as `vecMulVec u v`, and the inner product
`vᵀ A⁻¹ u` is `dotProduct v (A⁻¹.mulVec u)`.

## Proof sketch

Since `A` is invertible, factor `A + u vᵀ = A (I + (A⁻¹ u) vᵀ)`.
Set `w := A⁻¹ u`. By multiplicativity of the determinant,
`det(A + u vᵀ) = det(A) · det(I + w vᵀ)`.
Using `det_one_add_mul_comm` (which gives `det(I + col w · row v) = det(I + row v · col w)`),
the rank-one identity reduces `det(I + w vᵀ) = 1 + vᵀ w = 1 + vᵀ A⁻¹ u`,
and the result follows.
-/

open Matrix

/-- **Matrix determinant lemma**: For an invertible n×n real matrix `A` and vectors `u, v`,
`det(A + u vᵀ) = det(A) · (1 + vᵀ A⁻¹ u)`. -/
theorem root_aristotle (n : ℕ) (A : Matrix (Fin n) (Fin n) ℝ) (u v : Fin n → ℝ)
    (hA : IsUnit A.det) :
    (A + vecMulVec u v).det =
    A.det * (1 + dotProduct v (A⁻¹.mulVec u)) := by
  -- Factor A + uvᵀ = A * (I + (A⁻¹u)vᵀ).
  have h_factor : A + vecMulVec u v = A * (1 + vecMulVec (A⁻¹.mulVec u) v) := by
    norm_num [Matrix.mul_add, mul_vecMulVec]
    simp_all +decide
  simp_all +decide [Matrix.vecMulVec]
  -- Let w = A⁻¹u.
  set w := A⁻¹.mulVec u
  -- Rewrite 1 + wvᵀ as 1 + M * N where M is n×1 and N is 1×n.
  set M : Matrix (Fin n) (Fin 1) ℝ := fun i _ => w i
  set N : Matrix (Fin 1) (Fin n) ℝ := fun _ j => v j
  have hMN : 1 + of (fun x y => w x * v y) = 1 + M * N := by
    ext i j; simp +decide [Matrix.mul_apply]; rfl
  rw [hMN, Matrix.det_one_add_mul_comm]
  norm_num [dotProduct, mul_comm]
  rfl
