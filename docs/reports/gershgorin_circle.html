Gershgorin circle theorem

Gershgorin circle theorem. Let \(A=(a_{ij})\) be a complex \(n\times n\) matrix. For each row index \(i\), define the Gershgorin disc \[ D_i := \left\{ z \in \mathbb{C} : |z-a_{ii}| \le \sum_{j\ne i} |a_{ij}| \right\}. \] Then every eigenvalue \(\lambda\) of \(A\) lies in at least one of these discs. In other words, if \(Ax=\lambda x\) for some nonzero vector \(x\), then there exists an index \(r\) such that \[ |\lambda-a_{rr}| \le \sum_{j\ne r} |a_{rj}|. \] Hence \[ \operatorname{Spec}(A) \subseteq \bigcup_{i=1}^n D_i. \]

Informal statement:

For any complex \(n\times n\) matrix \(A\), its spectrum is contained in the union of its Gershgorin discs: \(\operatorname{Spec}(A) \subseteq \bigcup_{i=1}^n D_i\).

Informal proof:

Let \(\lambda\in\mathbb{C}\) be an eigenvalue of \(A\). By definition, there exists a nonzero vector \(x \in \mathbb{C}^n\) such that \(Ax=\lambda x\). Because \(x\neq 0\), at least one component of \(x\) is nonzero. Choose an index \(r\) such that \(|x_r| = \max_{1\le j\le n} |x_j|\). By our supporting lemma on the maximal eigenvector component, this choice of \(r\) guarantees that \(|\lambda-a_{rr}| \le \sum_{j\ne r} |a_{rj}|\). This is exactly the statement that \(\lambda \in D_r\), where \(D_r\) is the \(r\)-th Gershgorin disc. Since \(\lambda\) was an arbitrary eigenvalue of \(A\), we conclude that \(\operatorname{Spec}(A) \subseteq \bigcup_{i=1}^n D_i\).

/-- The `i`-th Gershgorin disc of a complex matrix `A` is
  `{z ∈ ℂ | ‖z - A i i‖ ≤ ∑ j ∈ univ.erase i, ‖A i j‖}`. -/
noncomputable def gershgorinDisc {n : ℕ} (A : Matrix (Fin n) (Fin n) ℂ) (i : Fin n) : Set ℂ :=
  {z : ℂ | ‖z - A i i‖ ≤ ∑ j ∈ univ.erase i, ‖A i j‖}

/-- The eigenvalues of a complex matrix `A`, defined as those `λ ∈ ℂ` for which there exists
  a nonzero vector `x` with `A.mulVec x = λ • x`. -/
def matrixEigenvalues {n : ℕ} (A : Matrix (Fin n) (Fin n) ℂ) : Set ℂ :=
  {eigval : ℂ | ∃ x : Fin n → ℂ, x ≠ 0 ∧ A.mulVec x = eigval • x}

private lemma exists_norm_max {n : ℕ} (x : Fin n → ℂ) (hx : x ≠ 0) :
    ∃ r : Fin n, ∀ j : Fin n, ‖x j‖ ≤ ‖x r‖ := by

Informal statement:

Let \(A=(a_{ij})\) be a complex \(n\times n\) matrix, and suppose \(Ax=\lambda x\) for a nonzero vector \(x\). If \(r\) is an index such that \(|x_r| = \max_{1\le j\le n} |x_j|\), then \(|\lambda-a_{rr}| \le \sum_{j\ne r} |a_{rj}|\).

Informal proof:

Look at the \(r\)-th coordinate of the eigenvalue equation \(Ax=\lambda x\). The \(r\)-th row gives \(\sum_{j=1}^n a_{rj}x_j = \lambda x_r\). Separate the diagonal term from the rest to obtain \(a_{rr}x_r + \sum_{j\ne r} a_{rj}x_j = \lambda x_r\). Rearranging, we get \((\lambda-a_{rr})x_r = \sum_{j\ne r} a_{rj}x_j\). Since \(x\neq 0\) and \(r\) maximizes \(|x_j|\), we must have \(x_r\neq 0\). Dividing by \(x_r\) yields \(\lambda-a_{rr} = \sum_{j\ne r} a_{rj}\frac{x_j}{x_r}\). Take absolute values and apply the triangle inequality: \(|\lambda-a_{rr}| \le \sum_{j\ne r} \left|a_{rj}\frac{x_j}{x_r}\right| = \sum_{j\ne r} |a_{rj}|\frac{|x_j|}{|x_r|}\). By the choice of \(r\), we have \(|x_j|\le |x_r|\) for every \(j\), meaning \(\frac{|x_j|}{|x_r|} \le 1\). Substituting this into the previous inequality yields \(|\lambda-a_{rr}| \le \sum_{j\ne r} |a_{rj}|\).
