Banach-Stone Theorem

Banach-Stone Theorem for real-valued continuous functions. Let \(X\) and \(Y\) be compact Hausdorff spaces. Suppose \[ T : C(X,\mathbb{R}) \to C(Y,\mathbb{R}) \] is a surjective linear isometry, where both spaces are equipped with the supremum norm. Then there exist: - a homeomorphism \(\tau : Y \to X\), and - a continuous function \(h : Y \to \{-1,1\}\), such that for every \(f \in C(X,\mathbb{R})\) and every \(y \in Y\), \[ (Tf)(y) = h(y)\, f(\tau(y)). \] In particular, \(X\) and \(Y\) are homeomorphic.

Informal statement:

Let $X$ and $Y$ be compact Hausdorff spaces. If $T : C(X,\mathbb{R}) \to C(Y,\mathbb{R})$ is a surjective linear isometry, then there exist a map $\tau : Y \to X$ and a function $h : Y \to \{-1,1\}$ such that $(Tf)(y) = h(y) f(\tau(y))$ for all $f \in C(X,\mathbb{R})$ and $y \in Y$.

Informal proof:

Let $T^* : C(Y,\mathbb{R})^* \to C(X,\mathbb{R})^*$ be the adjoint map, defined by $(T^*\mu)(f) = \mu(Tf)$. Since $T$ is a surjective linear isometry, $T^*$ is an isometric isomorphism between the dual spaces and maps the extreme points of the closed unit ball of $C(Y,\mathbb{R})^*$ bijectively onto the extreme points of the closed unit ball of $C(X,\mathbb{R})^*$. By the Riesz representation theorem, the dual space is identified with regular Borel measures, and the extreme points of the dual unit ball of $C(X,\mathbb{R})$ are exactly the evaluation functionals $\pm \delta_x$ for $x \in X$, and similarly for $Y$. Thus, for each $y \in Y$, the functional $\delta_y$ is an extreme point, so $T^* \delta_y = h(y) \delta_{\tau(y)}$ for some unique point $\tau(y) \in X$ and sign $h(y) \in \{-1,1\}$. Evaluating this identity at a function $f \in C(X,\mathbb{R})$ gives $(Tf)(y) = \delta_y(Tf) = (T^*\delta_y)(f) = h(y) f(\tau(y))$.

/-- Given T : C(X,ℝ) ≃ₗᵢ[ℝ] C(Y,ℝ), construct S with S(f)(y) = T(1)(y) * T(f)(y). -/
def twistedIsometry [Nonempty X] [Nonempty Y]
    (T : C(X, ℝ) ≃ₗᵢ[ℝ] C(Y, ℝ)) : C(X, ℝ) ≃ₗᵢ[ℝ] C(Y, ℝ) :=
  have hT1 : ∀ y, T 1 y = 1 ∨ T 1 y = -1 := isometry_image_one_values_aux T
  have hT1_sq : T 1 * T 1 = 1 := by
    ext y; simp [mul_apply]; rcases hT1 y with h | h <;> simp [h]
  LinearIsometryEquiv.mk
    ({ toFun := fun f => T 1 * T f
       invFun := fun g => T.symm (T 1 * g)
       left_inv := fun f => by
         have : T 1 * (T 1 * T f) = T f := by
           rw [← mul_assoc, hT1_sq, one_mul]
         simp [this]
       right_inv := fun g => by
         show T 1 * T (T.symm (T 1 * g)) = g
         rw [T.apply_symm_apply, ← mul_assoc, hT1_sq, one_mul]
       map_add' := fun f g => by simp [mul_add]
       map_smul' := fun r f => by simp [mul_comm] } : C(X, ℝ) ≃ₗ[ℝ] C(Y, ℝ))
    (fun f => by
      simp only [LinearEquiv.coe_mk]
      have h_abs : ∀ y, |T 1 y| = 1 := by
        intro y; rcases hT1 y with h | h <;> simp [h]
      conv_rhs => rw [← T.norm_map f]
      rw [norm_eq_iSup_norm, norm_eq_iSup_norm]
      congr 1; ext y; simp [mul_apply, Real.norm_eq_abs, h_abs])

lemma unital_isometry_mono [Nonempty X] [Nonempty Y]

    (hS1 : S 1 = 1)
    (f g : C(X, ℝ)) (hfg : ∀ x, f x ≤ g x) (y : Y) :

    · exfalso
      obtain ⟨f, hf⟩ := T.surjective (1 : C(Y, ℝ))
