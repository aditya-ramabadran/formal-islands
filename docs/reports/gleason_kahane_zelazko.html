Gleason-Kahane-Żelazko Theorem

Gleason-Kahane-Żelazko Theorem. Let \(A\) be a complex unital Banach algebra, and let \[ \phi : A \to \mathbb{C} \] be a complex-linear functional such that \[ \phi(1)=1. \] Assume that \(\phi\) does not vanish on invertible elements of \(A\); that is, \[ a \in A^\times \implies \phi(a) \ne 0. \] Then \(\phi\) is multiplicative: \[ \phi(xy)=\phi(x)\phi(y) \qquad\text{for all } x,y\in A. \] Equivalently, \(\phi\) is a nonzero algebra homomorphism from \(A\) to \(\mathbb{C}\).

Informal statement:

For every \(x \in A\), \(\phi(x^2) = \phi(x)^2\).

Informal proof:

Let \(x\in A\). Set \(a := x-\phi(x)1\). Then \(\phi(a)=\phi(x)-\phi(x)\phi(1)=0\), so \(a\in N\). Therefore \(\phi(a^2)=0\). Now expand: \[ a^2 = (x-\phi(x)1)^2 = x^2 - 2\phi(x)x + \phi(x)^2 1. \] Apply \(\phi\): \[ 0 = \phi(a^2) = \phi(x^2) - 2\phi(x)^2 + \phi(x)^2\phi(1). \] Since \(\phi(1)=1\), this simplifies to \[ 0 = \phi(x^2)-\phi(x)^2. \] So for every \(x\in A\), \(\phi(x^2)=\phi(x)^2\). Thus \(\phi\) preserves squares.

# Square Preservation

For every x in A, φ(x²) = φ(x)², where A is a complex unital Banach algebra
and φ : A → ℂ is a ℂ-linear functional with φ(1) = 1 satisfying
the kernel-square-zero property (∀ a, φ(a) = 0 → φ(a²) = 0).

This is a key step in the Gleason-Kahane-Żelazko theorem.

Proof sketch: Let x ∈ A. Set a := x - φ(x)·1. Then φ(a) = φ(x) - φ(x)·φ(1) = 0,

Hence φ(x²) = φ(x)².
-/

Polarization and Multiplicativity

Node id: polarization | Status: formal_verified

Used by (parent nodes): gkz_theorem

Formalization priority: 1. Rationale: A purely algebraic formal island that shows square preservation implies multiplicativity. Straightforward to formalize using elementary algebraic identities without requiring Banach algebra machinery.

Most recent formalization episode: verified the full parent node after 2 Lean verification attempts. Most recent formalization attempt verified the full parent node.

Informal statement:

If a linear functional \(\phi\) satisfies \(\phi(x^2) = \phi(x)^2\) for all \(x\), then \(\phi(xy) = \phi(x)\phi(y)\) for all \(x, y\).

Informal proof:

We now show that square preservation implies multiplicativity. For arbitrary \(x,y\in A\), apply the identity \(\phi(u^2)=\phi(u)^2\) to \(u=x+y\). Then \[ \phi((x+y)^2)=\phi(x+y)^2. \] Expanding both sides gives \[ \phi(x^2+xy+yx+y^2)=\phi(x)^2+2\phi(x)\phi(y)+\phi(y)^2. \] Using square preservation for \(x\) and \(y\), we obtain \[ \phi(xy+yx)=2\phi(x)\phi(y). \] Next apply the same identity to \(u=x+iy\). Then \[ \phi((x+iy)^2)=\phi(x+iy)^2. \] Expanding, \[ (x+iy)^2 = x^2 + i(xy+yx) - y^2, \] and so \[ \phi(x^2)+i\phi(xy+yx)-\phi(y^2) = \phi(x)^2 + 2i\phi(x)\phi(y) - \phi(y)^2. \] Again using square preservation, this reduces to the same relation \(\phi(xy+yx)=2\phi(x)\phi(y)\). To separate \(xy\) from \(yx\), we use the standard bilinear polarization identity in the scalar-valued setting: \[ xy = \frac14\bigl((x+y)^2-(x-y)^2\bigr) + \frac{1}{4i}\bigl((x+iy)^2-(x-iy)^2\bigr). \] Apply \(\phi\) to both sides. Since \(\phi\) preserves squares, \[ \phi(xy) = \frac14\Bigl(\phi(x+y)^2-\phi(x-y)^2\Bigr) + \frac{1}{4i}\Bigl(\phi(x+iy)^2-\phi(x-iy)^2\Bigr). \] Now compute this scalar expression in \(\mathbb{C}\). The right-hand side is exactly the polarization formula for multiplication in \(\mathbb{C}\), so it equals \(\phi(x)\phi(y)\). Thus \(\phi(xy)=\phi(x)\phi(y)\) for all \(x,y\in A\). Hence \(\phi\) is multiplicative.

If a ℂ-linear functional φ on a ℂ-algebra satisfies φ(x²) = φ(x)² for all x,
then φ is multiplicative: φ(xy) = φ(x)φ(y) for all x, y.

import Mathlib

open Complex

variable {A : Type*} [Ring A] [Algebra ℂ A]

/-- If φ preserves squares and φ(a) = 0, then φ(a*x) + φ(x*a) = 0.
Proof: expand φ((a+x)²) = φ(a+x)² and cancel using φ(a)=0 and square preservation. -/
lemma sq_pres_ker_anticomm (φ : A →ₗ[ℂ] ℂ) (hsq : ∀ x : A, φ (x ^ 2) = φ x ^ 2)
    (a x : A) (ha : φ a = 0) : φ (a * x) + φ (x * a) = 0 := by
  have h_expand : φ ((a + x) ^ 2) = φ (a + x) ^ 2 := hsq _
  simp_all +decide [sq, add_mul, mul_add]
  linear_combination' h_expand

/-- If φ preserves squares and φ(a) = 0, then φ(a*x) = 0 for all x.
Proof: from anticommutativity, φ(xa) = -φ(ax). Then φ((ax)²) + φ((xa)²) = 0
gives 2φ(ax)² = 0, hence φ(ax) = 0. -/
lemma sq_pres_ker_mul_left (φ : A →ₗ[ℂ] ℂ) (hsq : ∀ x : A, φ (x ^ 2) = φ x ^ 2)
    (a x : A) (ha : φ a = 0) : φ (a * x) = 0 := by
  have h1 : φ (a * x) + φ (x * a) = 0 :=
    sq_pres_ker_anticomm φ hsq a x ha
  have h3 : φ (a * (x * a * x)) + φ ((x * a * x) * a) = 0 :=
    sq_pres_ker_anticomm φ hsq a (x * a * x) ha
  simp_all +decide [sq, mul_assoc]
  grind

/-- Square preservation implies multiplicativity for a ℂ-linear functional on a ℂ-algebra.
For any x, y, set a = x - φ(x)·1 so that φ(a) = 0. Then φ(a·y) = 0 by the kernel lemma,
which expands to φ(x·y) - φ(x)·φ(y) = 0. -/
theorem polarization_aristotle (φ : A →ₗ[ℂ] ℂ) (hsq : ∀ x : A, φ (x ^ 2) = φ x ^ 2)
    (x y : A) : φ (x * y) = φ x * φ y := by
  set a : A := x - (φ x) • (1 : A)
  have ha : φ a = 0 := by
    simp +zetaDelta at *
    by_cases h : φ 1 = 0 <;> simp_all +decide [sq]
    · have := hsq (x + 1); simp_all +decide [add_mul, mul_add]
      linear_combination' this / 2
    · have := hsq 1; simp_all +decide [sq]
  have h_ay : φ (a * y) = 0 :=
    sq_pres_ker_mul_left φ hsq a y ha
  rw [sub_mul, smul_mul_assoc] at h_ay
  simp_all +decide [sub_eq_iff_eq_add]

Spectral Inclusion and Continuity

Node id: spectral_inclusion | Status: informal

Used by (parent nodes): kernel_square_zero

Informal statement:

For any \(a \in A\), \(\phi(a) \in \sigma(a)\). Consequently, \(\phi\) is continuous with \(\|\phi\| \le 1\).

Informal proof:

Let \(a\in A\). We claim that \(\phi(a) \in \sigma(a)\), where \(\sigma(a)\) denotes the spectrum of \(a\). Indeed, if \(\lambda\in\mathbb{C}\) is not in the spectrum of \(a\), then \(\lambda 1-a\) is invertible. By hypothesis, \(\phi(\lambda 1-a) \ne 0\). But by linearity and the normalization \(\phi(1)=1\), \[ \phi(\lambda 1-a)=\lambda\phi(1)-\phi(a)=\lambda-\phi(a). \] So \(\lambda-\phi(a)\ne 0\), i.e. \(\lambda\ne \phi(a)\). Thus \(\phi(a)\) cannot lie outside the spectrum, and therefore \(\phi(a)\in\sigma(a)\). Now every spectral value is bounded in modulus by the norm, so \[ |\phi(a)| \le r(a) \le \|a\|, \] where \(r(a)\) is the spectral radius. Hence \(\phi\) is continuous and \(\|\phi\| \le 1\). Since \(\phi(1)=1\) and \(\|1\|\ge 1\), in the usual normalized Banach-algebra setting \(\|1\|=1\), we get in fact \(\|\phi\|=1\).

Kernel Elements Have Square Zero

Node id: kernel_square_zero | Status: informal

Used by (parent nodes): square_preservation  |  Depends on (child nodes): spectral_inclusion

Informal statement:

For every \(a \in \ker \phi\), \(\phi(a^2) = 0\).

Informal proof:

Let \(N := \ker \phi\). We will prove that for every \(a\in N\), \(\phi(a^2)=0\). This is the key step. So fix \(a\in N\). Define an entire function \[ f : \mathbb{C} \to \mathbb{C}, \qquad f(z) := \phi(e^{za}). \] Because the exponential series converges in every Banach algebra and \(\phi\) is continuous linear, \(f\) is entire. For every \(z\in\mathbb{C}\), the element \(e^{za}\) is invertible, with inverse \(e^{-za}\). Therefore, by hypothesis, \[ f(z)=\phi(e^{za}) \ne 0 \qquad\text{for all } z\in\mathbb{C}. \] So \(f\) is a nowhere-vanishing entire function. Also, \(f(0)=\phi(1)=1\). Since \(a\in N\), \(f'(0)=\phi(a)=0\). Finally, because \(\|e^{za}\|\le e^{|z|\,\|a\|}\) and \(\|\phi\|\le 1\), we have the growth bound \[ |f(z)| \le e^{|z|\,\|a\|} \qquad\text{for all } z\in\mathbb{C}. \] Thus \(f\) is an entire function of exponential type, in particular of order at most \(1\). Now use the standard Hadamard factorization theorem for entire functions of finite order. A zero-free entire function of order at most \(1\) must be of the form \(f(z)=e^{\alpha z+\beta}\) for some constants \(\alpha,\beta\in\mathbb{C}\). Since \(f(0)=1\), we get \(1=f(0)=e^{\beta}\), so after choosing the obvious representative we may write \(\beta=0\). Hence \(f(z)=e^{\alpha z}\). Differentiating at \(0\), \(f'(0)=\alpha e^{0}=\alpha\). But \(f'(0)=0\), so \(\alpha=0\). Therefore \(f(z)=1\) for all \(z\in\mathbb{C}\). In particular, \(f''(0)=0\). On the other hand, differentiating the exponential series term-by-term shows that \(f''(0)=\phi(a^2)\). Hence \(\phi(a^2)=0\) for all \(a\in N\).
