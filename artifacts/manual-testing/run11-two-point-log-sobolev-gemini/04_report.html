Two-point logarithmic Sobolev inequality

Two-point logarithmic Sobolev inequality. Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), define the entropy of \(f^2\) by \[ \operatorname{Ent}_{\mu}(f^2) := \mathbb{E}_{\mu}[f^2 \log f^2] - \mathbb{E}_{\mu}[f^2] \log \bigl(\mathbb{E}_{\mu}[f^2]\bigr), \] with the convention \(0\log 0 := 0\). Then \[ \operatorname{Ent}_{\mu}(f^2) \le \frac{1}{2}\bigl(f(1)-f(-1)\bigr)^2. \] Equivalently, if \(a=f(1)\) and \(b=f(-1)\), then \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \] This is the logarithmic Sobolev inequality on the two-point Bernoulli space, and it is the basic finite-state model underlying tensorization arguments toward the Gaussian logarithmic Sobolev inequality.

Two-point logarithmic Sobolev inequality

Node id: two_point_lsi | Status: informal

Informal statement:

Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), define the entropy of \(f^2\) by \[ \operatorname{Ent}_{\mu}(f^2) := \mathbb{E}_{\mu}[f^2 \log f^2] - \mathbb{E}_{\mu}[f^2] \log \bigl(\mathbb{E}_{\mu}[f^2]\bigr), \] with the convention \(0\log 0 := 0\). Then \[ \operatorname{Ent}_{\mu}(f^2) \le \frac{1}{2}\bigl(f(1)-f(-1)\bigr)^2. \]

Informal proof:

Write \[ a := f(1), \qquad b := f(-1). \] Because \(\mu\) is the uniform measure on two points, the entropy inequality is exactly equivalent, after multiplying both sides by \(2\), to \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \] So it suffices to prove this scalar inequality. We first reduce to the case \(a,b\ge 0\). The left-hand side depends only on \(a^2\) and \(b^2\), so replacing \(a,b\) by \(|a|,|b|\) leaves it unchanged. On the other hand, \((|a|-|b|)^2 \le (a-b)^2\) by the reverse triangle inequality. Thus if the inequality holds for nonnegative values, it also holds for the original \(a,b\). Hence we may assume \(a,b\ge 0\). Next we use homogeneity to normalize. If we replace \((a,b)\) by \((ca,cb)\) with \(c\ge 0\), both sides of the inequality are multiplied by \(c^2\) (the extra \(\log(c^2)\) terms on the left cancel). Therefore the inequality is homogeneous of degree two. If \(a=b=0\), the claim is trivial. Otherwise let \(c = \sqrt{2/(a^2+b^2)}\). After scaling by \(c\), we may assume \(a^2+b^2=2\). Under this normalization, the desired inequality becomes \[ a^2\log(a^2)+b^2\log(b^2) \le (a-b)^2. \] Since \(a,b\ge 0\) and \(a^2+b^2=2\), there exists \(u\in[0,2]\) such that \[ u = a^2, \qquad b^2 = 2-u, \] so that \(a = \sqrt{u}\) and \(b = \sqrt{2-u}\). Thus it remains to prove that for every \(u\in[0,2]\), \[ u\log u + (2-u)\log(2-u) \le \bigl(\sqrt{u}-\sqrt{2-u}\bigr)^2. \] This scalar inequality holds by a separate lemma. Undoing the substitution \(u=a^2\), \(2-u=b^2\) and reversing the homogeneity and absolute value reductions yields the desired entropy bound.

Informal statement:

For every \(u\in[0,2]\), \[ u\log u + (2-u)\log(2-u) \le \bigl(\sqrt{u}-\sqrt{2-u}\bigr)^2, \] using the convention \(0\log 0 = 0\).

Informal proof:

Define \[ G(u) := \bigl(\sqrt{u}-\sqrt{2-u}\bigr)^2 - u\log u - (2-u)\log(2-u) \qquad (0\le u\le 2). \] We will show that \(G(u)\ge 0\) on \([0,2]\). First note that \(G\) is continuous on \([0,2]\) and smooth on \((0,2)\). Also \(G\) is symmetric under \(u\mapsto 2-u\). In particular, \[ G(1)=0, \] since \(\bigl(\sqrt1-\sqrt1\bigr)^2 = 0\) and \(1\log1 + 1\log1 = 0\). Now compute derivatives for \(0<u<2\). We have \(\bigl(\sqrt{u}-\sqrt{2-u}\bigr)^2 = 2 - 2\sqrt{u(2-u)}\), which gives \[ \frac{d}{du}\bigl(\sqrt{u}-\sqrt{2-u}\bigr)^2 = \frac{2(u-1)}{\sqrt{u(2-u)}}. \] Also, \[ \frac{d}{du}\bigl(u\log u + (2-u)\log(2-u)\bigr) = \log\!\left(\frac{u}{2-u}\right). \] Hence \[ G'(u)= \frac{2(u-1)}{\sqrt{u(2-u)}} - \log\!\left(\frac{u}{2-u}\right). \] Differentiating again yields \[ \frac{d}{du}\left(\frac{2(u-1)}{\sqrt{u(2-u)}}\right) = \frac{2}{\bigl(u(2-u)\bigr)^{3/2}}, \] while \[ \frac{d}{du}\log\!\left(\frac{u}{2-u}\right) = \frac1u + \frac{1}{2-u} = \frac{2}{u(2-u)}. \] Therefore \[ G''(u) = \frac{2}{\bigl(u(2-u)\bigr)^{3/2}} - \frac{2}{u(2-u)} = \frac{2\bigl(1-\sqrt{u(2-u)}\bigr)}{\bigl(u(2-u)\bigr)^{3/2}}. \] For \(u\in[0,2]\), we have \(u(2-u)=1-(u-1)^2 \le 1\), so \(\sqrt{u(2-u)}\le 1\). Hence \(G''(u)\ge 0\) for \(0<u<2\). Thus \(G\) is convex on \((0,2)\), and therefore on \([0,2]\) by continuity. Since \(G\) is symmetric under \(u\mapsto 2-u\), the point \(u=1\) is its center of symmetry, so \(G'(1)=0\). Because a convex function with a vanishing derivative at a point has a global minimum there, \(u=1\) is a global minimizer of \(G\). Therefore \[ G(u)\ge G(1)=0 \qquad\text{for all }u\in[0,2], \] which proves the inequality.

lemma alg_id (u : ℝ) (h0 : 0 < u) (h2 : u < 2) :

  have h2u : 0 ≤ 2 - u := by linarith
