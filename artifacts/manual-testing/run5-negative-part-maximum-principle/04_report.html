Weak maximum principle via the negative part

Node id: n0 | Status: informal

Display label: Weak maximum principle

Informal statement:

If \(-\Delta u \ge 0\) in \(\Omega\) and \(u \ge 0\) on \(\partial\Omega\), then \(u \ge 0\) in \(\Omega\).

Informal proof:

Test the inequality against the negative part \(u_-(x)=\max\{-u(x),0\}\). Since \(u_-\) vanishes on \(\partial\Omega\), integration by parts gives \(\int_\Omega (-\Delta u)u_-\,dx=\int_\Omega \nabla u\cdot\nabla u_-\,dx\ge 0\). On \(\{u<0\}\), \(\nabla u_-=-\nabla u\), while on \(\{u\ge 0\}\), \(\nabla u_-=0\), so \(\int_\Omega \nabla u\cdot\nabla u_-\,dx=-\int_\Omega |\nabla u_-|^2\,dx\). Hence \(\int_\Omega |\nabla u_-|^2\,dx\le 0\), so \(u_-\) is constant; the boundary condition forces that constant to be zero. Therefore \(u_\equiv 0\), i.e. \(u\ge 0\) in \(\Omega\).

Testing with the negative part and integration by parts

Node id: n1 | Status: candidate_formal

Display label: Test by \(u_-\)

Formalization priority: 2. Rationale: This packages the admissibility of \(u_-\) as a test function and the boundary-term cancellation, which is a standard but formally delicate step in weak maximum principle proofs.

Informal statement:

For \(u_-(x)=\max\{-u(x),0\}\), one has \(u_-\ge 0\) in \(\Omega\), \(u_-=0\) on \(\partial\Omega\), and \[ \int_\Omega (-\Delta u)\,u_-\,dx = \int_\Omega \nabla u\cdot\nabla u_-\,dx. \]

Informal proof:

Because \(u\ge 0\) on \(\partial\Omega\), the negative part \(u_-\) has zero boundary trace. Thus it is an admissible test function for the weak form of \(-\Delta u \ge 0\), and integration by parts removes the boundary term.

Energy identity for the negative part

Node id: n2 | Status: formal_failed

Display label: Negative-part identity

Formalization priority: 1. Rationale: This is the main local formal island: the pointwise case split on \(u\ge 0\) vs. \(u<0\) produces the decisive energy identity \(\int \nabla u\cdot\nabla u_- = -\int |\nabla u_-|^2\).

Informal statement:

The gradients satisfy \[ \int_\Omega \nabla u\cdot\nabla u_-\,dx = -\int_\Omega |\nabla u_-|^2\,dx. \]

Informal proof:

On \(\{u\ge 0\}\), one has \(u_-=0\) and hence \(\nabla u_-=0\). On \(\{u<0\}\), \(u_-=-u\), so \(\nabla u_-=-\nabla u\). Therefore \(\nabla u\cdot\nabla u_-=-|\nabla u_-|^2\) pointwise where \(u<0\), and the identity follows after integrating over \(\Omega\).

Vanishing of the negative part

Node id: n3 | Status: candidate_formal

Display label: Conclude \(u_-=0\)

Formalization priority: 3. Rationale: This is the final deduction from the nonpositive energy integral to \(u_-\equiv 0\), combining nonnegativity, constancy, and the boundary trace condition.

Informal statement:

If \(\int_\Omega |\nabla u_-|^2\,dx\le 0\), then \(u_-=0\) in \(\Omega\), hence \(u\ge 0\) in \(\Omega\).

Informal proof:

Since \(|\nabla u_-|^2\ge 0\), the integral inequality forces \(|\nabla u_-|^2=0\) almost everywhere. Thus \(u_-\) is constant on \(\Omega\); the boundary condition \(u_-=0\) on \(\partial\Omega\) forces that constant to be zero. Consequently \(u_-\equiv 0\), so \(u\) never takes negative values.
