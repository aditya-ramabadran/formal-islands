Pinsker's inequality via reduction to the Bernoulli case

Pinsker's inequality via reduction to the Bernoulli case. Let \((X,\mathcal F)\) be a measurable space, and let \(P\) and \(Q\) be probability measures on \((X,\mathcal F)\). Assume \(P\ll Q\), so that the Kullback–Leibler divergence \[ D_{\mathrm{KL}}(P\|Q) := \int_X \log\!\left(\frac{dP}{dQ}\right)\,dP \] is well defined in \([0,\infty]\). Then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2, \] where \[ \|P-Q\|_{\mathrm{TV}}:=\sup_{A\in\mathcal F}|P(A)-Q(A)|. \] Equivalently, if \(\delta:=\|P-Q\|_{\mathrm{TV}}\), then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2. \] The key local scalar step is the Bernoulli inequality \[ p\log\!\left(\frac{p}{q}\right)+(1-p)\log\!\left(\frac{1-p}{1-q}\right)\ge 2(p-q)^2, \qquad 0<p,q<1, \] which appears after collapsing the general measures to a two-point partition.

Pinsker's Inequality

Node id: root | Status: informal

Display label: Pinsker: $D_{\mathrm{KL}}(P\|Q)\ge 2\|P-Q\|_{\mathrm{TV}}^2$

Informal statement:

Let \ $(X,\mathcal{F})$ be a measurable space and $P,Q$ probability measures with $P\ll Q$. Then $D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2$.

Informal proof:

Set $\delta:=\|P-Q\|_{\mathrm{TV}}$. By a Hahn-decomposition argument, choose $A\in\mathcal{F}$ with $P(A)-Q(A)=\delta\ge 0$. Put $p:=P(A)$, $q:=Q(A)$, and define $T:=\mathbf{1}_A:X\to\{0,1\}$. The pushforwards satisfy $T_\#P=(p,1-p)$ and $T_\#Q=(q,1-q)$, so $D_{\mathrm{KL}}(T_\#P\|T_\#Q)=p\log(p/q)+(1-p)\log((1-p)/(1-q))$. By the data-processing inequality (monotonicity of KL under pushforward), $D_{\mathrm{KL}}(P\|Q)\ge D_{\mathrm{KL}}(T_\#P\|T_\#Q)$. Applying the Bernoulli scalar inequality then gives $D_{\mathrm{KL}}(T_\#P\|T_\#Q)\ge 2(p-q)^2=2\delta^2$. Hence $D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2=2\|P-Q\|_{\mathrm{TV}}^2$.

Data-Processing Inequality for KL Divergence

Node id: dpi | Status: informal

Display label: DPI: $D_{\mathrm{KL}}(P\|Q)\ge D_{\mathrm{KL}}(T_\#P\|T_\#Q)$

Informal statement:

Let $T:(X,\mathcal{F})\to(Y,\mathcal{G})$ be a measurable map. If $P\ll Q$, then $D_{\mathrm{KL}}(P\|Q)\ge D_{\mathrm{KL}}(T_\#P\|T_\#Q)$. In particular, for $T=\mathbf{1}_A$, $D_{\mathrm{KL}}(P\|Q)\ge p\log(p/q)+(1-p)\log((1-p)/(1-q))$ where $p=P(A)$, $q=Q(A)$.

Informal proof:

This is a standard measure-theoretic fact: coarsening the observation space by a measurable map can only decrease relative entropy. Concretely, if $f=dP/dQ$ is the Radon–Nikodym derivative, the log-sum inequality (Jensen applied to the convex function $t\mapsto t\log t$) gives $D_{\mathrm{KL}}(T_\#P\|T_\#Q)=\int_Y\log\frac{d(T_\#P)}{d(T_\#Q)}\,d(T_\#P)\le\int_X\log f\,dP=D_{\mathrm{KL}}(P\|Q)$. For the binary map $T=\mathbf{1}_A$ this specialises immediately.

Bernoulli Scalar Inequality

Node id: bernoulli_ineq | Status: informal

Display label: Bernoulli: $p\log\frac{p}{q}+(1-p)\log\frac{1-p}{1-q}\ge 2(p-q)^2$

Informal statement:

For all $p,q\in(0,1)$, $p\log\!\ \left(\tfrac{p}{q}\right)+(1-p)\log\!\left(\tfrac{1-p}{1-q}\right)\ge 2(p-q)^2$.

Informal proof:

Fix $q\in(0,1)$ and define $F_q(p):=p\log(p/q)+(1-p)\log((1-p)/(1-q))-2(p-q)^2$ for $p\in(0,1)$. We show $F_q\ge 0$. First, $F_q(q)=0$. By the convexity of $F_q$ (proved separately) and the critical-point computation $F_q'(q)=\log(q/q)-\log((1-q)/(1-q))-4(q-q)=0$, the point $p=q$ is a global minimizer of $F_q$. Hence $F_q(p)\ge F_q(q)=0$ for all $p\in(0,1)$.

Convexity of $F_q$ via Second-Derivative Bound

Node id: convexity_fq | Status: candidate_formal

Display label: $F_q''(p)=\frac{1}{p(1-p)}-4\ge 0$

Formalization priority: 2. Rationale: The second-derivative computation $F_q''(p)=1/(p(1-p))-4\ge 0$ is the key technical lemma supporting the Bernoulli inequality. It is a concrete, checkable real-analysis fact (AM–GM for $p(1-p)\le 1/4$) and is a natural sub-lemma to formalize separately before assembling the full Bernoulli node.

Informal statement:

For fixed $q\in(0,1)$, the function $F_q(p):=p\log(p/q)+(1-p)\log((1-p)/(1-q))-2(p-q)^2$ satisfies $F_q''(p)=\tfrac{1}{p(1-p)}-4\ge 0$ for all $p\in(0,1)$, so $F_q$ is convex.

Informal proof:

Differentiating twice: $\frac{d}{dp}[p\log(p/q)]=\log(p/q)+1$ and $\frac{d}{dp}[(1-p)\log((1-p)/(1-q))]=-\log((1-p)/(1-q))-1$, so $F_q'(p)=\log\frac{p(1-q)}{q(1-p)}-4(p-q)$. Differentiating again: $F_q''(p)=\frac{1}{p}+\frac{1}{1-p}-4=\frac{1}{p(1-p)}-4$. The elementary AM–GM inequality gives $p(1-p)\le\left(\frac{p+(1-p)}{2}\right)^2=\tfrac{1}{4}$ for $p\in(0,1)$, so $\frac{1}{p(1-p)}\ge 4$, hence $F_q''(p)\ge 0$. Therefore $F_q$ is convex on $(0,1)$.

$F_q$ vanishes at $p = q$

Node id: bernoulli_ineq_refined_local_claim | Status: formal_verified

Display label: Fq_zero_at_q

Formalization priority: 1. Rationale: This establishes the baseline value needed to convert the global-minimiser conclusion into the inequality $F_q(p)\ge 0$. It is narrower and more atomic than the full Bernoulli claim, isolating the boundary-value computation as a standalone verifiable step.

Informal statement:

For fixed $q\in(0,1)$, the function $F_q(p):=p\log(p/q)+(1-p)\log((1-p)/(1-q))-2(p-q)^2$ satisfies $F_q(q)=0$.

Informal proof:

Substitute $p=q$: $q\log(q/q)+(1-q)\log((1-q)/(1-q))-2(q-q)^2=q\cdot 0+(1-q)\cdot 0-0=0$.
