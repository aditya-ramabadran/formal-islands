Harmonic functions minimize Dirichlet energy among competitors with the same boundary values

Harmonic functions minimize the Dirichlet energy among competitors with the same boundary values. Let \(\Omega \subset \mathbb{R}^d\) be a bounded domain with sufficiently regular boundary, and let \(g\) be prescribed boundary data. Suppose \(u\) is a sufficiently regular function such that: - \(u=g\) on \(\partial\Omega\), - and \(u\) is weakly harmonic, meaning \[ \int_\Omega \nabla u \cdot \nabla \varphi \, dx = 0 \qquad \text{for every } \varphi \in H_0^1(\Omega). \] Then for every sufficiently regular \(v\) with the same boundary data \(v=g\) on \(\partial\Omega\), one has \[ \int_\Omega |\nabla u|^2\,dx \le \int_\Omega |\nabla v|^2\,dx. \] Moreover, \[ \int_\Omega |\nabla v|^2\,dx = \int_\Omega |\nabla u|^2\,dx + \int_\Omega |\nabla(v-u)|^2\,dx. \]

Display label: Dirichlet principle for weakly harmonic \(u\)

Formalization priority: 2. Rationale: Natural end-to-end theorem statement once the energy identity is available; formalization is straightforward from nonnegativity of the final term.

Informal statement:

If \(u\) is weakly harmonic in \(\Omega\) and \(u=g\) on \(\partial\Omega\), then for every sufficiently regular \(v\) with \(v=g\) on \(\partial\Omega\), \[ \int_\Omega |\nabla v|^2\,dx = \int_\Omega |\nabla u|^2\,dx + \int_\Omega |\nabla(v-u)|^2\,dx, \] and hence \[ \int_\Omega |\nabla u|^2\,dx \le \int_\Omega |\nabla v|^2\,dx. \]

Informal proof:

Fix a competitor \(v\) with the same boundary data and set \(w:=v-u\). By the common boundary condition, \(w=0\) on \(\partial\Omega\), so \(w\in H_0^1(\Omega)\). Apply the energy decomposition from the local claim to obtain \[ \int_\Omega |\nabla v|^2\,dx = \int_\Omega |\nabla u|^2\,dx + \int_\Omega |\nabla(v-u)|^2\,dx. \] Since the last term is nonnegative, the minimizing inequality follows immediately. Thus \(u\) minimizes the Dirichlet energy among all competitors with the same boundary values.

Energy decomposition using the weak harmonicity test \(\varphi=v-u\)

Node id: n_energy_split | Status: informal

Display label: Energy identity

Informal statement:

For \(w:=v-u\), if \(u\) and \(v\) have the same boundary values, then \(w\in H_0^1(\Omega)\) and \[ \int_\Omega |\nabla v|^2\,dx = \int_\Omega |\nabla u|^2\,dx + \int_\Omega |\nabla w|^2\,dx. \]

Informal proof:

Set \(w:=v-u\). Since \(u=v=g\) on \(\partial\Omega\), we have \(w=0\) on \(\partial\Omega\), hence \(w\in H_0^1(\Omega)\). Also \(\nabla v=\nabla(u+w)=\nabla u+\nabla w\), so pointwise \[ |\nabla v|^2=|\nabla u+\nabla w|^2=|\nabla u|^2+2\nabla u\cdot\nabla w+|\nabla w|^2. \] Integrating gives \[ \int_\Omega |\nabla v|^2\,dx = \int_\Omega |\nabla u|^2\,dx +2\int_\Omega \nabla u\cdot\nabla w\,dx +\int_\Omega |\nabla w|^2\,dx. \] Because \(w\in H_0^1(\Omega)\), it is an admissible test function in the weak harmonicity condition for \(u\), so \[ \int_\Omega \nabla u\cdot\nabla w\,dx=0. \] Therefore the cross term vanishes and the stated identity follows. Replacing \(w\) by \(v-u\) yields the advertised form.

theorem n_energy_split
    (d : ℕ)
    (Ω : Set (EuclideanSpace ℝ (Fin d)))
    (grad_u grad_w : EuclideanSpace ℝ (Fin d) → EuclideanSpace ℝ (Fin d))
    (hu : Integrable (fun x => ‖grad_u x‖ ^ 2) (volume.restrict Ω))
    (hw : Integrable (fun x => ‖grad_w x‖ ^ 2) (volume.restrict Ω))
    (huw : Integrable (fun x => @inner ℝ _ _ (grad_u x) (grad_w x)) (volume.restrict Ω))
    (horth : ∫ x, @inner ℝ _ _ (grad_u x) (grad_w x) ∂(volume.restrict Ω) = 0) :
    (∫ x, ‖grad_u x + grad_w x‖ ^ 2 ∂(volume.restrict Ω)) =
      (∫ x, ‖grad_u x‖ ^ 2 ∂(volume.restrict Ω)) +
        ∫ x, ‖grad_w x‖ ^ 2 ∂(volume.restrict Ω) := by
  have h_expand :
      (fun x => ‖grad_u x + grad_w x‖ ^ 2) =
        fun x => ‖grad_u x‖ ^ 2 + 2 * @inner ℝ _ _ (grad_u x) (grad_w x) + ‖grad_w x‖ ^ 2 := by
    funext x
    simpa using norm_add_sq_real (grad_u x) (grad_w x)
  rw [h_expand]
  have h_two_inner :
      Integrable (fun x => 2 * @inner ℝ _ _ (grad_u x) (grad_w x)) (volume.restrict Ω) :=
    huw.const_mul (2 : ℝ)
  have h_first_two :
      Integrable (fun x => ‖grad_u x‖ ^ 2 + 2 * @inner ℝ _ _ (grad_u x) (grad_w x))
        (volume.restrict Ω) :=
    hu.add h_two_inner
  have h_assoc :
      (fun x => ‖grad_u x‖ ^ 2 + 2 * @inner ℝ _ _ (grad_u x) (grad_w x) + ‖grad_w x‖ ^ 2) =
        fun x => (‖grad_u x‖ ^ 2 + 2 * @inner ℝ _ _ (grad_u x) (grad_w x)) + ‖grad_w x‖ ^ 2 := by
    funext x
    ring
  rw [h_assoc, integral_add h_first_two hw, integral_add hu h_two_inner,
    integral_const_mul, horth]
  ring

end FormalIslands
