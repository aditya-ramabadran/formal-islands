Pinsker's inequality via reduction to the Bernoulli case

Pinsker's inequality via reduction to the Bernoulli case. Let \((X,\mathcal F)\) be a measurable space, and let \(P\) and \(Q\) be probability measures on \((X,\mathcal F)\). Assume \(P\ll Q\), so that the Kullback–Leibler divergence \[ D_{\mathrm{KL}}(P\|Q) := \int_X \log\!\left(\frac{dP}{dQ}\right)\,dP \] is well defined in \([0,\infty]\). Then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2, \] where \[ \|P-Q\|_{\mathrm{TV}}:=\sup_{A\in\mathcal F}|P(A)-Q(A)|. \] Equivalently, if \(\delta:=\|P-Q\|_{\mathrm{TV}}\), then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2. \] The key local scalar step is the Bernoulli inequality \[ p\log\!\left(\frac{p}{q}\right)+(1-p)\log\!\left(\frac{1-p}{1-q}\right)\ge 2(p-q)^2, \qquad 0<p,q<1, \] which appears after collapsing the general measures to a two-point partition.

Display label: Pinsker's inequality: $D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2$

Informal statement:

Let $P,Q$ be probability measures on $(X,\mathcal{F})$ with $P\ll Q$. Then $D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2$.

Informal proof:

Set $\delta:=\|P-Q\|_{\mathrm{TV}}$. By the Hahn decomposition choose $A\in\mathcal{F}$ with $P(A)-Q(A)=\delta\ge 0$. Set $p:=P(A)$, $q:=Q(A)$, and let $T:=\mathbf{1}_A:X\to\{0,1\}$. The pushforward measures satisfy $T_\#P=(p,1-p)$ and $T_\#Q=(q,1-q)$, so $D_{\mathrm{KL}}(T_\#P\|T_\#Q)=p\log(p/q)+(1-p)\log((1-p)/(1-q))$. By the data-processing inequality, $D_{\mathrm{KL}}(P\|Q)\ge D_{\mathrm{KL}}(T_\#P\|T_\#Q)$. By the Bernoulli scalar inequality, $D_{\mathrm{KL}}(T_\#P\|T_\#Q)\ge 2(p-q)^2=2\delta^2$. Chaining the two inequalities gives $D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2=2\,\|P-Q\|_{\mathrm{TV}}^2$.

Bernoulli scalar inequality

Node id: bernoulli_ineq | Status: informal

Display label: Bernoulli inequality: $p\log\frac{p}{q}+(1-p)\log\frac{1-p}{1-q}\ge 2(p-q)^2$

Informal statement:

For all $p,q\in(0,1)$, \[p\log\!\left(\frac{p}{q}\right)+(1-p)\log\!\left(\frac{1-p}{1-q}\right)\ge 2(p-q)^2.\]

Informal proof:

Fix $q\in(0,1)$ and define $F_q(p):=p\log(p/q)+(1-p)\log((1-p)/(1-q))-2(p-q)^2$ for $p\in(0,1)$. We show $F_q(p)\ge 0$. **Value at $p=q$:** $F_q(q)=0$. **First derivative:** $F_q'(p)=\log(p/q)-\log((1-p)/(1-q))-4(p-q)$, so $F_q'(q)=0$. **Second derivative:** $F_q''(p)=\frac{1}{p}+\frac{1}{1-p}-4=\frac{1}{p(1-p)}-4$. Since $p(1-p)\le\tfrac{1}{4}$ for all $p\in(0,1)$ (AM-GM), we have $\frac{1}{p(1-p)}\ge 4$, so $F_q''(p)\ge 0$. **Conclusion:** $F_q$ is convex and $p=q$ is a critical point with $F_q(q)=0$, hence a global minimizer. Therefore $F_q(p)\ge 0$ for all $p\in(0,1)$.
