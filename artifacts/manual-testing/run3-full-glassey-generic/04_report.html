Full Glassey blow-up proof

Glassey blow-up argument for the focusing mass-critical nonlinear Schrodinger equation. Consider a sufficiently regular solution \(u(t,x)\) to the focusing mass-critical NLS on \(\mathbb{R}^d\): \[ i\partial_t u+\Delta u=-|u|^{4/d}u. \] Assume: - \(u_0\in H^1(\mathbb{R}^d)\), - \(|x|u_0\in L^2(\mathbb{R}^d)\), - the energy \(E(u_0)\) is negative, where \[ E(u)=\frac12\int_{\mathbb{R}^d}|\nabla u|^2\,dx -\frac{1}{2+4/d}\int_{\mathbb{R}^d}|u|^{2+4/d}\,dx. \] Then the maximal forward existence time \(T\) is finite and \[ \|u(t)\|_{H^1(\mathbb{R}^d)}\to\infty \quad\text{as }t\uparrow T. \]

Glassey blow-up for focusing mass-critical NLS

Node id: thm_glassey_blowup | Status: informal

Display label: Main theorem

Informal statement:

Under the assumptions \(u_0\in H^1\), \(|x|u_0\in L^2\), and \(E(u_0)<0\), the forward maximal lifespan \(T\) of the focusing mass-critical NLS solution is finite and \(\|u(t)\|_{H^1(\mathbb{R}^d)}\to\infty\) as \(t\uparrow T\).

Informal proof:

Define the variance \(V(t)=\int |x|^2|u(t,x)|^2\,dx\). The virial computation gives \(V''(t)=16E(u_0)<0\), so \(V\) is strictly concave downward and must reach \(0\) in finite forward time; hence for some finite \(T_*\), \(V(t)\to0\) as \(t\uparrow T_*\). Applying the weighted inequality to \(u(t)\) and using mass conservation yields \(\|u_0\|_2^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_2\), so \(\|\nabla u(t)\|_2\to\infty\) as \(t\uparrow T_*\). Since \(\|u(t)\|_2\) is conserved, also \(\|u(t)\|_{H^1}\to\infty\). By the standard \(H^1\) continuation principle, finite-time \(H^1\)-norm blow-up forces the maximal forward lifespan to be finite, and taking \(T\) to be that maximal forward time gives the theorem.

Virial identity with negative constant second derivative

Rationale: This is the main self-contained technical step: an explicit virial/energy identity proved by differentiating the variance twice and carrying out nontrivial integration-by-parts and nonlinear coefficient bookkeeping. It discharges substantial inferential burden for the downstream concavity argument, while avoiding broader PDE continuation machinery and avoiding the lower-value elementary concavity consequence.

Informal statement:

For \(V(t)=\int_{\mathbb{R}^d}|x|^2|u(t,x)|^2\,dx\), one has \(V''(t)=16E(u(t))=16E(u_0)<0\).

Informal proof:

Differentiate \(V\) once and substitute \(u_t=i\Delta u+i|u|^{4/d}u\). The term \(\int |x|^2|u|^{2+4/d}\,dx\) is real, so after multiplication by \(i\) its real part vanishes. Integrating by parts in the Laplacian term gives \(V'(t)=4\operatorname{Im}\int x\cdot\nabla u\,\overline u\,dx\). Differentiate again, substitute formulas for \(u_t\) and \(\overline{u_t}\), and split linear and nonlinear pieces. A standard integration-by-parts computation gives the linear contribution \(8\int |\nabla u|^2\,dx\). For the nonlinear contribution, rewrite the real-part expression so that after subtracting the matching term one is left with \(4\int x\cdot \nabla\!\left(\frac{d}{2d+4}|u|^{2+4/d}\right)dx\), and integrating by parts using \(\operatorname{div}(x)=d\) yields \(-\frac{16}{2+4/d}\int |u|^{2+4/d}\,dx\). Thus \(V''(t)=8\int |\nabla u|^2\,dx-\frac{16}{2+4/d}\int |u|^{2+4/d}\,dx=16E(u(t))\). Energy conservation then gives \(V''(t)=16E(u_0)<0\).

Weighted \(L^2\) versus gradient inequality

Node id: weighted_uncertainty | Status: informal

Display label: Weighted inequality

Informal statement:

For sufficiently regular \(f\), \[\|f\|_{L^2(\mathbb{R}^d)}^2\le \frac{2}{d}\,\|x f\|_{L^2(\mathbb{R}^d)}\,\|\nabla f\|_{L^2(\mathbb{R}^d)}.\]

Informal proof:

Use \(\operatorname{div}(x)=d\): \(\|f\|_2^2=\frac1d\int (\operatorname{div}x)|f|^2\,dx=-\frac1d\int x\cdot \nabla(|f|^2)\,dx=-\frac{2}{d}\operatorname{Re}\int x\cdot \nabla f\,\overline f\,dx\). Then apply \(|\operatorname{Re} z|\le |z|\) and Cauchy-Schwarz to obtain the stated bound.

Negative virial curvature forces finite-time vanishing of the variance

Node id: variance_collapse | Status: informal

Display label: Concavity of variance

Informal statement:

If \(V''(t)=16E(u_0)<0\), then \(V(t)\ge0\) cannot hold for all large positive times. Hence there exists a finite \(T_*>0\) such that \(V(T_*)=0\), equivalently \(V(t)\to0\) as \(t\uparrow T_*\).

Informal proof:

Because \(V''\) is a negative constant, \(V\) is a concave quadratic-type function with negative leading coefficient. Such a function eventually becomes negative as \(t\to+\infty\). Since by definition \(V(t)=\int |x|^2|u(t,x)|^2\,dx\ge0\), this is impossible unless the graph meets \(0\) at some finite forward time. Taking the first such time and using continuity yields \(V(t)\to0\) as \(t\uparrow T_*\).

Variance collapse implies gradient and \(H^1\) blow-up

Node id: gradient_blowup | Status: informal

Display label: Blow-up from variance collapse

Informal statement:

Let \(M=\|u_0\|_{L^2(\mathbb{R}^d)}>0\). Then for all times of existence, \[M^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)}.\] Therefore, if \(V(t)\to0\) as \(t\uparrow T_*\), then \(\|\nabla u(t)\|_2\to\infty\), and since \(\|u(t)\|_2=M\) is conserved, also \(\|u(t)\|_{H^1}\to\infty\).

Informal proof:

Apply the weighted inequality with \(f=u(t)\). Mass conservation gives \(\|u(t)\|_2=\|u_0\|_2=:M\), and \(M>0\) since \(E(u_0)<0\) rules out the zero solution. Using \(\|xu(t)\|_2^2=V(t)\), one obtains \(M^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_2\). As \(t\uparrow T_*\), the left side stays fixed while \(\sqrt{V(t)}\to0\), so necessarily \(\|\nabla u(t)\|_2\to\infty\). Finally, \(\|u(t)\|_{H^1}^2=\|u(t)\|_2^2+\|\nabla u(t)\|_2^2\) and \(\|u(t)\|_2=M\) is conserved, hence \(\|u(t)\|_{H^1}\to\infty\).

Standard \(H^1\) continuation criterion

Node id: continuation_principle | Status: informal

Display label: Continuation principle

Informal statement:

For the mass-critical NLS in \(H^1(\mathbb{R}^d)\), if a solution on \([0,T_*)\) satisfies \(\sup_{t<T_*}\|u(t)\|_{H^1}<\infty\), then it extends beyond \(T_*\). Equivalently, finite-time failure of continuation forces \(\|u(t)\|_{H^1}\to\infty\) as \(t\uparrow T_*\).

Informal proof:

This is the standard local well-posedness and blow-up alternative in \(H^1\) for the mass-critical nonlinear Schr\"odinger equation. It is used only to identify the finite blow-up time produced by the virial argument with the maximal forward lifespan.
