Itô's formula for a twice continuously differentiable function of Brownian motion

Itô's formula for a twice continuously differentiable function of Brownian motion. Let \((B_t)_{t\ge 0}\) be a standard one-dimensional Brownian motion, let \(T>0\), and let \(f:\mathbb{R}\to\mathbb{R}\) be a \(C^2\) function. Assume for simplicity that \(f'\) and \(f''\) are bounded and that \(f''\) is uniformly continuous on \(\mathbb{R}\). Then for every \(t\in[0,T]\), \[ f(B_t) = f(B_0) + \int_0^t f'(B_s)\,dB_s + \frac12\int_0^t f''(B_s)\,ds. \]

Itô's Formula

Node id: ito_formula | Status: informal

Dependency nodes: taylor_expansion, mn_vanishes, rn_vanishes

Informal statement:

For $(B_t)_{t\ge 0}$ a standard Brownian motion and $f\in C^2(\mathbb{R})$ with $f',f''$ bounded and $f''$ uniformly continuous, for every $t\in[0,T]$: \[ f(B_t) = f(B_0) + \int_0^t f'(B_s)\,dB_s + \frac{1}{2}\int_0^t f''(B_s)\,ds. \]

Informal proof:

Fix $t\in[0,T]$ and a partition $0=t_0<t_1<\cdots<t_n=t$ with $\Delta B_k := B_{t_{k+1}}-B_{t_k}$ and $\Delta t_k := t_{k+1}-t_k$. By the telescoping identity and Taylor's theorem (see node taylor_expansion), \[ f(B_t)-f(B_0) = \sum_{k=0}^{n-1} f'(B_{t_k})\,\Delta B_k + \frac{1}{2}\sum_{k=0}^{n-1} f''(B_{t_k})(\Delta B_k)^2 + R_n. \] The three terms are handled separately: (1) **First term** converges to $\int_0^t f'(B_s)\,dB_s$ in $L^2$ by the definition and $L^2$-isometry of the Itô integral for adapted left-endpoint processes. (2) **Second term**: Write $\frac{1}{2}\sum_k f''(B_{t_k})(\Delta B_k)^2 = \frac{1}{2}\sum_k f''(B_{t_k})\,\Delta t_k + \frac{1}{2}M_n$, where $M_n := \sum_k f''(B_{t_k})((\Delta B_k)^2-\Delta t_k)$. The Riemann sum $\sum_k f''(B_{t_k})\,\Delta t_k$ converges a.s. to $\int_0^t f''(B_s)\,ds$ since $s\mapsto f''(B_s)$ is a.s. continuous. By node mn_vanishes, $M_n\to 0$ in $L^2$. Hence the second term converges in probability to $\frac{1}{2}\int_0^t f''(B_s)\,ds$. (3) **Third term** $R_n\to 0$ in probability by node rn_vanishes. Taking the mesh to zero, all three limits combine to give Itô's formula.

Taylor Expansion Identity with Remainder Bound

Node id: taylor_expansion | Status: formal_failed

Parent nodes: ito_formula, rn_vanishes

Display label: Taylor identity & remainder bound

Formalization priority: 1. Rationale: Entirely deterministic, self-contained, and the pivotal step of the whole proof. The telescoping sum plus pointwise Taylor identity plus explicit remainder bound is a clean finite algebraic statement with no measure theory, making it the highest-yield formal island: formalizing it in Lean would validate the heart of the argument at low stochastic overhead.

Informal statement:

For each increment, Taylor's theorem gives \[ f(B_{t_{k+1}}) = f(B_{t_k}) + f'(B_{t_k})\,\Delta B_k + \tfrac{1}{2}f''(B_{t_k})(\Delta B_k)^2 + \tfrac{1}{2}(f''(\xi_k)-f''(B_{t_k}))(\Delta B_k)^2 \] for some $\xi_k$ between $B_{t_k}$ and $B_{t_{k+1}}$. Summing via the telescoping identity yields \[ f(B_t)-f(B_0) = \sum_{k=0}^{n-1}f'(B_{t_k})\,\Delta B_k + \frac{1}{2}\sum_{k=0}^{n-1}f''(B_{t_k})(\Delta B_k)^2 + R_n, \] where the remainder satisfies \[ |R_n| \le \frac{1}{2}\,\omega\!\left(\max_k |\Delta B_k|\right)\sum_{k=0}^{n-1}(\Delta B_k)^2, \] and $\omega$ is any modulus of continuity for $f''$.

Informal proof:

This is purely deterministic. Apply the second-order Taylor formula with Lagrange remainder to $f$ at each point $B_{t_k}$: there exists $\xi_k$ with $|\xi_k - B_{t_k}| \le |\Delta B_k|$ such that \[ f(B_{t_{k+1}}) - f(B_{t_k}) = f'(B_{t_k})\,\Delta B_k + \tfrac{1}{2}f''(B_{t_k})(\Delta B_k)^2 + \tfrac{1}{2}(f''(\xi_k)-f''(B_{t_k}))(\Delta B_k)^2. \] Sum over $k=0,\ldots,n-1$ and use the telescoping identity $\sum_k (f(B_{t_{k+1}})-f(B_{t_k})) = f(B_t)-f(B_0)$. Define \[ R_n := \frac{1}{2}\sum_{k=0}^{n-1}(f''(\xi_k)-f''(B_{t_k}))(\Delta B_k)^2. \] Since $|\xi_k - B_{t_k}|\le |\Delta B_k|$, we have $|f''(\xi_k)-f''(B_{t_k})|\le \omega(|\Delta B_k|)\le \omega(\max_j|\Delta B_j|)$ for any modulus $\omega$ of $f''$. Hence \[ |R_n| \le \frac{1}{2}\,\omega\!\left(\max_k |\Delta B_k|\right)\sum_{k=0}^{n-1}(\Delta B_k)^2. \qquad\square \]

/-
The telescoping Taylor expansion identity with remainder bound.

For a C² function f and a sequence of points x₀, ..., xₙ, we have:
  f(xₙ) - f(x₀) = Σ f'(xₖ)·Δxₖ + (1/2)·Σ f''(xₖ)·(Δxₖ)² + R
where the remainder R satisfies |R| ≤ (1/2)·ω(δ)·Σ(Δxₖ)² for any δ bounding max|Δxₖ|.
-/
theorem taylor_expansion_aristotle
    (n : ℕ) (f : ℝ → ℝ) (x : ℕ → ℝ) (hf : ContDiff ℝ 2 f)
    (ω : ℝ → ℝ) (hω_nonneg : ∀ t, 0 ≤ ω t) (hω_mono : Monotone ω)
    (hω_mod : ∀ u v, |deriv (deriv f) u - deriv (deriv f) v| ≤ ω (|u - v|))
    (δ : ℝ) (hδ : ∀ k, k < n → |x (k + 1) - x k| ≤ δ) :
    ∃ R : ℝ,
      f (x n) - f (x 0) =
        (∑ k ∈ Finset.range n, deriv f (x k) * (x (k + 1) - x k))
        + 1 / 2 * (∑ k ∈ Finset.range n, deriv (deriv f) (x k) * (x (k + 1) - x k) ^ 2)
        + R
      ∧ |R| ≤ 1 / 2 * ω δ * ∑ k ∈ Finset.range n, (x (k + 1) - x k) ^ 2 := by
  refine' ⟨ _, eq_add_of_sub_eq' rfl, _ ⟩;
  -- Apply the single-step bound to each term in the sum.
  have h_remainder_bound : ∀ k < n, |(f (x (k + 1)) - f (x k) - deriv f (x k) * (x (k + 1) - x k) - 1 / 2 * deriv (deriv f) (x k) * (x (k + 1) - x k) ^ 2)| ≤ 1 / 2 * ω (|x (k + 1) - x k|) * (x (k + 1) - x k) ^ 2 := by
    exact fun k a => taylor_second_order_remainder_bound f (x k) (x (k + 1)) hf ω hω_nonneg hω_mono hω_mod;
  -- By the triangle inequality, we can bound the absolute value of the sum.
  have h_triangle : |∑ k ∈ Finset.range n, (f (x (k + 1)) - f (x k) - deriv f (x k) * (x (k + 1) - x k) - 1 / 2 * deriv (deriv f) (x k) * (x (k + 1) - x k) ^ 2)| ≤ ∑ k ∈ Finset.range n, (1 / 2 * ω δ) * (x (k + 1) - x k) ^ 2 := by
    exact le_trans ( Finset.abs_sum_le_sum_abs _ _ ) ( Finset.sum_le_sum fun i hi => le_trans ( h_remainder_bound i ( Finset.mem_range.mp hi ) ) ( by gcongr ; aesop ) );
  convert h_triangle using 1 <;> norm_num [ Finset.mul_sum _ _ _, mul_assoc ];
  rw [ ← Finset.sum_sub_distrib ] ; rw [ Finset.sum_range_sub ( fun i => f ( x i ) ) ] ; ring;

end

# Quadratic Variation Error Vanishes in L²

This file formalizes the key deterministic estimate underlying the proof that the
quadratic variation error vanishes in L² in the proof of Itô's formula for Brownian
motion.

## Setting

In the Itô formula proof, one defines
  `M_n := ∑_{k=0}^{n-1} f''(B_{t_k}) · ((ΔB_k)² - Δt_k)`.

By martingale orthogonality (the cross terms vanish because each increment is
independent of the past), we have
  `𝔼[M_n²] = ∑_k 𝔼[f''(B_{t_k})² · ((ΔB_k)² - Δt_k)²]`.

Using `|f''| ≤ C` and the Gaussian fourth-moment identity
  `𝔼[((ΔB_k)² - Δt_k)²] = 2(Δt_k)²`,
we obtain the bound
  `𝔼[M_n²] ≤ 2C² ∑_k (Δt_k)²`.

We formalize below the **deterministic** bound
  `2C² ∑_k (Δt_k)² ≤ 2C² · mesh · T`,
which shows that the right-hand side tends to 0 as the mesh tends to 0.

## Main result

`mn_vanishes_aristotle`: For a partition of `[0, T]` with non-negative increments
`Δt k` summing to `T`, mesh at most `δ`, and `C ≥ 0`:
  `2 * C² * ∑ k, (Δt k)² ≤ 2 * C² * δ * T`.
-/
