Pinsker's inequality via reduction to the Bernoulli case

Pinsker's inequality via reduction to the Bernoulli case. Let \((X,\mathcal F)\) be a measurable space, and let \(P\) and \(Q\) be probability measures on \((X,\mathcal F)\). Assume \(P\ll Q\), so that the Kullback–Leibler divergence \[ D_{\mathrm{KL}}(P\|Q) := \int_X \log\!\left(\frac{dP}{dQ}\right)\,dP \] is well defined in \([0,\infty]\). Then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\,\|P-Q\|_{\mathrm{TV}}^2, \] where \[ \|P-Q\|_{\mathrm{TV}}:=\sup_{A\in\mathcal F}|P(A)-Q(A)|. \] Equivalently, if \(\delta:=\|P-Q\|_{\mathrm{TV}}\), then \[ D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2. \] The key local scalar step is the Bernoulli inequality \[ p\log\!\left(\frac{p}{q}\right)+(1-p)\log\!\left(\frac{1-p}{1-q}\right)\ge 2(p-q)^2, \qquad 0<p,q<1, \] which appears after collapsing the general measures to a two-point partition.

Pinsker's inequality

Node id: pinsker_main | Status: informal

Informal statement:

Let \((X,\mathcal F)\) be a measurable space, and let \(P\) and \(Q\) be probability measures on \((X,\mathcal F)\). Assume \(P\ll Q\). Then \(D_{\mathrm{KL}}(P\|Q)\ge 2\|P-Q\|_{\mathrm{TV}}^2\).

Informal proof:

Let \(\delta:=\|P-Q\|_{\mathrm{TV}}=\sup_{A\in\mathcal F}|P(A)-Q(A)|\). We will prove \(D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2\). Choose a measurable set \(A\in\mathcal F\) such that \(|P(A)-Q(A)|=\delta\). Replacing \(A\) by its complement if necessary, we may assume \(P(A)-Q(A)=\delta\ge 0\). Set \(p:=P(A)\) and \(q:=Q(A)\). Then \(p-q=\delta\). Define the measurable map \(T:X\to\{0,1\}\) by \(T(x)=\mathbf 1_A(x)\). The pushforward measures on \(\{0,1\}\) are \(T_\#P=(p,1-p)\) and \(T_\#Q=(q,1-q)\). By the monotonicity of relative entropy under measurable pushforward (the data-processing inequality), we have \[ D_{\mathrm{KL}}(P\|Q) \ge D_{\mathrm{KL}}(T_\#P\|T_\#Q) = p\log\!\left(\frac pq\right) + (1-p)\log\!\left(\frac{1-p}{1-q}\right). \] By the Bernoulli inequality for relative entropy, this lower bound is at least \(2(p-q)^2\). Since \(p-q=\delta\), we conclude \(D_{\mathrm{KL}}(P\|Q)\ge 2\delta^2\), completing the proof.

Bernoulli inequality for relative entropy

Node id: bernoulli_inequality | Status: informal

Informal statement:

For \(0<p,q<1\), \[ p\log\!\left(\frac pq\right) + (1-p)\log\!\left(\frac{1-p}{1-q}\right) \ge 2(p-q)^2. \]

Informal proof:

Fix \(q\in(0,1)\), and define for \(p\in(0,1)\) \[ F_q(p) := p\log\!\left(\frac{p}{q}\right) + (1-p)\log\!\left(\frac{1-p}{1-q}\right) - 2(p-q)^2. \] We will show that \(F_q(p)\ge 0\) for all \(p\in(0,1)\). Evaluating at \(p=q\) yields \(F_q(q) = 0\). Now compute the first derivative with respect to \(p\): \[ F_q'(p) = \log\!\left(\frac{p(1-q)}{q(1-p)}\right) - 4(p-q). \] Evaluating the first derivative at \(p=q\) gives \(F_q'(q) = 0\). Differentiating again, we find the second derivative: \[ F_q''(p) = \frac1p + \frac{1}{1-p} - 4 = \frac{1}{p(1-p)} - 4. \] For \(0<p<1\), the elementary inequality \(p(1-p)\le \frac14\) holds, which implies \(\frac{1}{p(1-p)}\ge 4\). Therefore, \(F_q''(p)\ge 0\) for all \(p\in(0,1)\), meaning \(F_q\) is convex on \((0,1)\). Since \(p=q\) is a critical point of the convex function \(F_q\), it is a global minimizer. Since \(F_q(q)=0\), it follows that \(F_q(p)\ge 0\) for all \(p\in(0,1)\), which is exactly the claimed inequality.

For any real number \(p\) such that \(0 < p < 1\), the inequality \(4 \le \frac{1}{p(1-p)}\) holds.

Informal proof:

Because \(0 < p < 1\), the product \(p(1-p)\) is strictly positive. Taking the reciprocal of both sides of \(0 < p(1-p) \le \frac{1}{4}\) yields \(\frac{1}{p(1-p)} \ge 4\).
