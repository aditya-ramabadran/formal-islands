Two-point logarithmic Sobolev inequality

Two-point logarithmic Sobolev inequality. Let \(\mu\) be the uniform probability measure on the two-point space \(\{-1,1\}\). For a function \(f : \{-1,1\} \to \mathbb{R}\), define the entropy of \(f^2\) by \[ \operatorname{Ent}_{\mu}(f^2) := \mathbb{E}_{\mu}[f^2 \log f^2] - \mathbb{E}_{\mu}[f^2] \log \bigl(\mathbb{E}_{\mu}[f^2]\bigr), \] with the convention \(0\log 0 := 0\). Then \[ \operatorname{Ent}_{\mu}(f^2) \le \frac{1}{2}\bigl(f(1)-f(-1)\bigr)^2. \] Equivalently, if \(a=f(1)\) and \(b=f(-1)\), then \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \] This is the logarithmic Sobolev inequality on the two-point Bernoulli space, and it is the basic finite-state model underlying tensorization arguments toward the Gaussian logarithmic Sobolev inequality.

Two-point logarithmic Sobolev inequality

Node id: n0 | Status: informal

Display label: Two-point LSI

Informal statement:

For \(f : \{-1,1\} \to \mathbb{R}\) and \(a=f(1), b=f(-1)\), one has \[ \operatorname{Ent}_{\mu}(f^2) \le \tfrac12 (a-b)^2, \] equivalently \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \]

Informal proof:

Expand the entropy under the uniform measure on \(\{-1,1\}\) to obtain the scalar inequality in \(a,b\). The left side depends only on \(a^2,b^2\), while \((|a|-|b|)^2\le (a-b)^2\), so it is enough to treat \(a,b\ge 0\). The inequality is homogeneous of degree \(2\), hence after excluding the trivial case \(a=b=0\) we may scale so that \(a^2+b^2=2\). Writing \(u=a^2\in[0,2]\) and \(b^2=2-u\), the claim becomes \[ u\log u + (2-u)\log(2-u) \le (\sqrt u-\sqrt{2-u})^2. \] Define \[ G(u):=(\sqrt u-\sqrt{2-u})^2-u\log u-(2-u)\log(2-u). \] By the convexity analysis from the auxiliary node, \(G\) is convex on \((0,2)\), and \(G'(1)=0\). Hence \(u=1\) is a global minimizer. Since also \(G(1)=0\), we get \(G(u)\ge 0\) for all \(u\in[0,2]\). Undoing the substitution and scaling yields the scalar inequality for all real \(a,b\), hence the entropy inequality.

Entropy expansion and scalar reduction

Node id: n1 | Status: candidate_formal

Display label: Scalar reduction

Formalization priority: 3. Rationale: Low-risk algebraic bridge from entropy notation to the scalar inequality; helpful but more routine than the convexity lemma.

Informal statement:

If \(a=f(1)\) and \(b=f(-1)\), then \[ \operatorname{Ent}_{\mu}(f^2) = \frac12\Bigl(a^2\log(a^2)+b^2\log(b^2)-(a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right)\Bigr). \] Therefore \[ \operatorname{Ent}_{\mu}(f^2)\le \frac12(a-b)^2 \] is equivalent to the scalar inequality \[ a^2 \log(a^2) + b^2 \log(b^2) - (a^2+b^2)\log\!\left(\frac{a^2+b^2}{2}\right) \le (a-b)^2. \]

Informal proof:

Because \(\mu\) is uniform on two points, \[ \mathbb E_\mu[f^2]=\frac{a^2+b^2}{2},\qquad \mathbb E_\mu[f^2\log f^2]=\frac{a^2\log(a^2)+b^2\log(b^2)}{2}. \] Substitute these into the definition of entropy and simplify. Multiplying the desired entropy inequality by \(2\) gives exactly the displayed scalar inequality.

Sign reduction and homogeneous normalization

Node id: n2 | Status: candidate_formal

Display label: Normalization

Formalization priority: 2. Rationale: Useful local reduction packaging two nontrivial proof devices, sign reduction and homogeneity normalization, that are easy to formalize and materially simplify the main goal.

Informal statement:

To prove the scalar inequality, it suffices to consider \(a,b\ge 0\) with normalization \(a^2+b^2=2\). Under this normalization, setting \(u=a^2\in[0,2]\) and \(b^2=2-u\) reduces the goal to \[ u\log u + (2-u)\log(2-u) \le (\sqrt{u}-\sqrt{2-u})^2. \]

Informal proof:

The left-hand side depends only on \(a^2,b^2\), so replacing \((a,b)\) by \((|a|,|b|)\) leaves it unchanged; meanwhile \((|a|-|b|)^2\le (a-b)^2\), so the nonnegative case implies the general case. Next, both sides scale by \(c^2\) under \((a,b)\mapsto(ca,cb)\) for \(c\ge 0\): the extra \(\log(c^2)\) terms cancel on the entropy side. Thus, unless \(a=b=0\), one may choose \(c=\sqrt{2/(a^2+b^2)}\) and assume \(a^2+b^2=2\). Then write \(u=a^2\), so \(b^2=2-u\), and because \(a,b\ge 0\), we have \(a=\sqrt u\), \(b=\sqrt{2-u}\). This yields the one-variable inequality.

Convexity of the one-variable function

Node id: n3 | Status: formal_verified

Display label: Convexity lemma

Formalization priority: 1. Rationale: Best formal island: a concrete one-variable calculus lemma with explicit derivatives and a clean convexity-to-minimum conclusion.

Informal statement:

Define \[ G(u):=(\sqrt{u}-\sqrt{2-u})^2-u\log u-(2-u)\log(2-u),\qquad 0\le u\le 2, \] with \(0\log 0:=0\). Then \(G\) is continuous on \([0,2]\), \(C^2\) on \((0,2)\), satisfies \(G(1)=0\) and \(G'(1)=0\), and \[ G''(u)=\frac{2\bigl(1-\sqrt{u(2-u)}\bigr)}{(u(2-u))^{3/2}}\ge 0 \qquad (0<u<2). \] Hence \(G\) is convex and attains its global minimum at \(u=1\), so \(G(u)\ge 0\) on \([0,2]\).

Informal proof:

Use \[ (\sqrt{u}-\sqrt{2-u})^2=2-2\sqrt{u(2-u)} \] to compute \[ \frac{d}{du}(\sqrt{u}-\sqrt{2-u})^2=\frac{2(u-1)}{\sqrt{u(2-u)}}. \] Also, \[ \frac{d}{du}\bigl(u\log u+(2-u)\log(2-u)\bigr)=\log\!\left(\frac{u}{2-u}\right). \] Therefore \[ G'(u)=\frac{2(u-1)}{\sqrt{u(2-u)}}-\log\!\left(\frac{u}{2-u}\right). \] Differentiating again gives \[ \frac{d}{du}\left(\frac{2(u-1)}{\sqrt{u(2-u)}}\right)=\frac{2}{(u(2-u))^{3/2}}, \qquad \frac{d}{du}\log\!\left(\frac{u}{2-u}\right)=\frac{2}{u(2-u)}. \] Hence \[ G''(u)=\frac{2}{(u(2-u))^{3/2}}-\frac{2}{u(2-u)} =\frac{2(1-\sqrt{u(2-u)})}{(u(2-u))^{3/2}}. \] Since \(u(2-u)=1-(u-1)^2\le 1\) on \([0,2]\), we have \(G''(u)\ge 0\). Thus \(G\) is convex. Directly, \(G(1)=0\) and \(G'(1)=0\), so by convexity \(u=1\) is a global minimizer and \(G(u)\ge 0\) for all \(u\in[0,2]\).
