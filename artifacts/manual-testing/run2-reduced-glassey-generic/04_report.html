Reduced Glassey blow-up argument

Finite-time blow-up from a virial identity for focusing mass-critical NLS. Let \(u(t,x)\) be a sufficiently regular solution on \(\mathbb{R}^d\) with finite variance. Assume: 1. mass is conserved: \[ \|u(t)\|_{L^2(\mathbb{R}^d)}=\|u_0\|_{L^2(\mathbb{R}^d)}=:M>0, \] 2. energy is conserved and negative: \[ E(u(t))=E(u_0)<0, \] 3. the variance \[ V(t)=\int_{\mathbb{R}^d}|x|^2|u(t,x)|^2\,dx \] satisfies \[ V''(t)=16E(u_0)<0, \] 4. for every sufficiently regular \(f\), \[ \|f\|_{L^2(\mathbb{R}^d)}^2 \le \frac{2}{d}\,\|x f\|_{L^2(\mathbb{R}^d)}\,\|\nabla f\|_{L^2(\mathbb{R}^d)}. \] Then there exists \(T<\infty\) such that \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \]

Finite-time blow-up from virial collapse of the variance

Node id: thm_blowup | Status: informal

Display label: Blow-up theorem

Informal statement:

There exists \(T<\infty\) such that \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \]

Informal proof:

Use the estimate \[ M^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)} \] from mass conservation and the weighted inequality, together with the existence of a finite time \(T>0\) such that \(V(t)\to 0\) as \(t\uparrow T\). Since \(M>0\) is fixed, the right-hand side can stay bounded below by \(M^2\) only if \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \] Thus the solution blows up in finite positive time.

Mass/variance-gradient estimate

Node id: n1 | Status: informal

Display label: \(M^2\lesssim \sqrt{V}\,\|\nabla u\|_{L^2}\)

Informal statement:

For every time in the interval of existence, \[ M^2=\|u(t)\|_{L^2(\mathbb{R}^d)}^2 \le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)}. \]

Informal proof:

Apply the assumed weighted inequality \[ \|f\|_{L^2(\mathbb{R}^d)}^2\le \frac{2}{d}\,\|xf\|_{L^2(\mathbb{R}^d)}\,\|\nabla f\|_{L^2(\mathbb{R}^d)} \] with \(f=u(t)\). Mass conservation gives \(\|u(t)\|_{L^2}=M\), and by definition of the variance, \[ \|xu(t)\|_{L^2(\mathbb{R}^d)}^2=V(t), \] so \(\|xu(t)\|_{L^2}=\sqrt{V(t)}\). Substituting yields the claimed estimate.

Negative virial curvature forces finite-time collapse of the variance

Node id: n2 | Status: informal

Display label: Finite-time variance collapse

Informal statement:

There exists a finite time \(T>0\) such that \[ V(T)=0, \] and hence \[ V(t)\to 0\quad\text{as }t\uparrow T. \]

Informal proof:

From the virial identity and \(E(u_0)<0\), \[ V''(t)=16E(u_0)<0 \] for all times of existence. Integrating twice gives \[ V(t)=8E(u_0)t^2+at+b \] for some constants \(a,b\), so the coefficient of \(t^2\) is negative. Therefore \(V(t)\to -\infty\) as \(t\to +\infty\). On the other hand, \[ V(t)=\int_{\mathbb{R}^d}|x|^2|u(t,x)|^2\,dx\ge 0 \] whenever the solution exists. Hence \(V\) cannot remain positive for all \(t\ge 0\). By continuity, there is a finite \(T>0\) with \(V(T)=0\), equivalently \(V(t)\to 0\) as \(t\uparrow T\).
