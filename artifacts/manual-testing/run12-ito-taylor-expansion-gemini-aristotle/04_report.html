Itô's formula (Main Theorem)

Node id: ito_formula | Status: informal

Informal statement:

Let $(B_t)_{t\ge 0}$ be a standard one-dimensional Brownian motion, let $T>0$, and let $f:\mathbb{R}\to\mathbb{R}$ be a $C^2$ function. Assume that $f'$ and $f''$ are bounded and that $f''$ is uniformly continuous on $\mathbb{R}$. Then for every $t\in[0,T]$, \[ f(B_t) = f(B_0) + \int_0^t f'(B_s)\,dB_s + \frac12\int_0^t f''(B_s)\,ds. \]

Informal proof:

Fix $t\in[0,T]$ and a partition $0=t_0<t_1<\cdots<t_n=t$. Write $\Delta B_k := B_{t_{k+1}}-B_{t_k}$. By the deterministic Taylor expansion identity, we have \[ f(B_t)-f(B_0) = \sum_{k=0}^{n-1} f'(B_{t_k})\,\Delta B_k + \frac12 \sum_{k=0}^{n-1} f''(B_{t_k})(\Delta B_k)^2 + R_n, \] where the remainder satisfies $|R_n| \le \frac12\,\omega\!\left(\max_k |\Delta B_k|\right)\sum_{k=0}^{n-1}(\Delta B_k)^2$, with $\omega$ being a modulus of continuity for $f''$. We analyze the three terms as the mesh of the partition tends to zero. First, the left-endpoint sums converge in $L^2$ (and hence in probability) to the Itô integral $\int_0^t f'(B_s)\,dB_s$. Second, the quadratic variation sums converge in probability to $\frac12\int_0^t f''(B_s)\,ds$. Third, since Brownian paths are almost surely uniformly continuous on $[0,t]$, $\max_k |\Delta B_k| \to 0$ almost surely. Furthermore, the quadratic variation $\sum_{k=0}^{n-1}(\Delta B_k)^2$ converges to $t$ in probability. Hence $R_n \to 0$ in probability. Putting everything together, the sum converges to the terms in Itô's formula.

Informal statement:

Let $f:\mathbb{R}\to\mathbb{R}$ be a $C^2$ function and let $\omega$ be a modulus of continuity for $f''$. For any finite sequence of real numbers $B_{t_0}, B_{t_1}, \dots, B_{t_n}$, let $\Delta B_k = B_{t_{k+1}}-B_{t_k}$. Then \[ f(B_{t_n})-f(B_{t_0}) = \sum_{k=0}^{n-1} f'(B_{t_k})\,\Delta B_k + \frac12 \sum_{k=0}^{n-1} f''(B_{t_k})(\Delta B_k)^2 + R_n, \] where the remainder $R_n$ satisfies \[ |R_n| \le \frac12\,\omega\!\left(\max_{0 \le k \le n-1} |\Delta B_k|\right)\sum_{k=0}^{n-1}(\Delta B_k)^2. \]

Informal proof:

We begin with the telescoping identity $f(B_{t_n})-f(B_{t_0}) = \sum_{k=0}^{n-1}\bigl(f(B_{t_{k+1}})-f(B_{t_k})\bigr)$. Applying Taylor's theorem with remainder to each increment, there exists $\xi_k$ between $B_{t_k}$ and $B_{t_{k+1}}$ such that \[ f(B_{t_{k+1}}) = f(B_{t_k}) + f'(B_{t_k})\,\Delta B_k + \frac12 f''(B_{t_k}) (\Delta B_k)^2 + \frac12\bigl(f''(\xi_k)-f''(B_{t_k})\bigr)(\Delta B_k)^2. \] Summing over $k$, we obtain the stated identity with remainder \[ R_n = \frac12\sum_{k=0}^{n-1}\bigl(f''(\xi_k)-f''(B_{t_k})\bigr)(\Delta B_k)^2. \] Since $\xi_k$ lies between $B_{t_k}$ and $B_{t_{k+1}}$, we have $|\xi_k - B_{t_k}| \le |\Delta B_k|$. By the definition of the modulus of continuity, $|f''(\xi_k)-f''(B_{t_k})| \le \omega(|\Delta B_k|) \le \omega(\max_j |\Delta B_j|)$. The bound on $|R_n|$ then follows from the triangle inequality.

/-! ### Main theorem: deterministic Taylor expansion for increments -/

/-- The maximum absolute increment of a sequence B over indices 0..n-1. -/
def maxAbsIncrement (B : ℕ → ℝ) (n : ℕ) : ℝ :=
  Finset.fold max 0 (fun k => |B (k + 1) - B k|) (Finset.range n)

/-
Each |ΔBₖ| is at most the maximum absolute increment.
-/
lemma le_maxAbsIncrement (B : ℕ → ℝ) (n : ℕ) (k : ℕ) (hk : k < n) :
    |B (k + 1) - B k| ≤ maxAbsIncrement B n := by
      -- By definition of maxAbsIncrement, we know that |B (k + 1) - B k| ≤ maxAbsIncrement B n for any k < n.
      have h_max : ∀ n, ∀ k < n, |B (k + 1) - B k| ≤ (Finset.range n).fold max 0 (fun k => |B (k + 1) - B k|) := by
        intro n k hk; induction' hk with n h IH <;> simp_all +decide [ Finset.range_add_one ] ;
      exact h_max n k hk

/-
**Deterministic Taylor expansion for increments.**

Let f : ℝ → ℝ be a C² function and ω a modulus of continuity for f''.
For any finite sequence B : ℕ → ℝ and any n, there exists R such that

  f(B n) - f(B 0) = Σ_{k<n} f'(B k) · ΔBₖ + ½ Σ_{k<n} f''(B k) · (ΔBₖ)² + R

and |R| ≤ ½ · ω(maxΔ) · Σ_{k<n} (ΔBₖ)², where maxΔ = max_{k<n} |ΔBₖ|.
-/
theorem deterministic_taylor_expansion_aristotle
    (f : ℝ → ℝ) (hf : ContDiff ℝ 2 f)
    (ω : ℝ → ℝ) (hω_nonneg : ∀ x, 0 ≤ x → 0 ≤ ω x) (hω_mono : Monotone ω)
    (hω_mod : ∀ x y, |iteratedDeriv 2 f x - iteratedDeriv 2 f y| ≤ ω (|x - y|))
    (n : ℕ) (B : ℕ → ℝ) :
    ∃ R : ℝ,
      f (B n) - f (B 0) =
        ∑ k ∈ range n, deriv f (B k) * (B (k+1) - B k) +
        1 / 2 * ∑ k ∈ range n, iteratedDeriv 2 f (B k) * (B (k+1) - B k) ^ 2 + R ∧
      |R| ≤ 1 / 2 * ω (maxAbsIncrement B n) *
        ∑ k ∈ range n, (B (k+1) - B k) ^ 2 := by
          refine' ⟨ _, _, _ ⟩;
          exact ∑ k ∈ Finset.range n, ( f ( B ( k + 1 ) ) - f ( B k ) - deriv f ( B k ) * ( B ( k + 1 ) - B k ) - 1 / 2 * iteratedDeriv 2 f ( B k ) * ( B ( k + 1 ) - B k ) ^ 2 );
          · exact Nat.recOn n ( by norm_num ) fun n ih => by norm_num [ Finset.sum_range_succ ] at * ; linarith;
          · rw [ Finset.mul_sum _ _ _ ];
            refine' le_trans ( Finset.abs_sum_le_sum_abs _ _ ) ( Finset.sum_le_sum fun i hi => _ );
            refine' le_trans ( taylor_single_step_bound f hf ω hω_mono hω_mod ( B i ) ( B ( i + 1 ) ) ) _;
            gcongr;
            exact hω_mono ( le_maxAbsIncrement B n i ( Finset.mem_range.mp hi ) )

end

Informal statement:

Let $(B_t)_{t\ge 0}$ be a standard Brownian motion and $f''$ be a bounded continuous function. For a partition $0=t_0<t_1<\cdots<t_n=t$, let $\Delta B_k = B_{t_{k+1}}-B_{t_k}$ and $\Delta t_k = t_{k+1}-t_k$. As the mesh of the partition tends to zero, \[ \frac12 \sum_{k=0}^{n-1} f''(B_{t_k})(\Delta B_k)^2 \to \frac12\int_0^t f''(B_s)\,ds \] in probability.

Informal proof:

We can split the sum as follows: \[ \frac12 \sum_{k=0}^{n-1} f''(B_{t_k})(\Delta B_k)^2 = \frac12 \sum_{k=0}^{n-1} f''(B_{t_k})\,\Delta t_k + \frac12 M_n, \] where $M_n := \sum_{k=0}^{n-1} f''(B_{t_k})\bigl((\Delta B_k)^2-\Delta t_k\bigr)$. For the first term, since $s\mapsto f''(B_s)$ is almost surely continuous, the Riemann sums converge almost surely to $\int_0^t f''(B_s)\,ds$. For the second term, observe that $M_n$ is a sum of martingale differences. Let $C$ be a bound on $|f''|$. This file formalizes the key deterministic estimates underlying the convergence
of the quadratic variation term in Itô's formula for Brownian motion.

## Main results

The informal target theorem states: for a partition 0 = t₀ < t₁ < ⋯ < tₙ = t
and a bounded continuous function f'', as the mesh → 0,
  ½ ∑ f''(B_{tₖ})(ΔBₖ)² → ½ ∫₀ᵗ f''(Bₛ) ds    in probability.

The proof decomposes this into:
1. A Riemann‐sum part: ∑ f''(B_{tₖ}) Δtₖ → ∫₀ᵗ f''(Bₛ) ds  (a.s.)
2. A martingale remainder: Mₙ = ∑ f''(B_{tₖ})((ΔBₖ)² − Δtₖ) → 0  in L²

The L² convergence of the remainder is controlled by the deterministic bound
  E[Mₙ²] ≤ 2C² ∑ (Δtₖ)²  ≤  2C² · mesh · t → 0.

We formalize:
- `partition_sum_eq_endpoint`: telescoping — ∑ Δtₖ = t
- `partition_sum_sq_le_mesh_mul_length`: ∑(Δtₖ)² ≤ mesh · t
- `quadratic_variation_split`: the algebraic decomposition of ∑ gₖ aₖ²
- `quadratic_variation_convergence_aristotle`: the main L² bound,
  stated abstractly over any probability space, showing that the weighted sum of
  centered squared increments has variance bounded by 2C² · mesh · t.

/-- The k-th increment of a sequence of partition points. -/
def partIncrement (pts : ℕ → ℝ) (k : ℕ) : ℝ := pts (k + 1) - pts k

/-- The mesh of a partition with n subintervals (n > 0). -/
def partMesh (pts : ℕ → ℝ) (n : ℕ) : ℝ :=
  if h : 0 < n then
    Finset.sup' (Finset.range n) (by simp [Nat.pos_iff_ne_zero.mp h]) fun k => partIncrement pts k
  else 0

/-- A well-formed partition of [0, t] with n subintervals. -/

  mono : Monotone pts
  start_eq : pts 0 = 0
  end_eq : pts n = t
