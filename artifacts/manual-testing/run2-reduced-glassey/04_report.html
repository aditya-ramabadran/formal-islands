Reduced Glassey blow-up argument

Finite-time blow-up from a virial identity for focusing mass-critical NLS. Let \(u(t,x)\) be a sufficiently regular solution on \(\mathbb{R}^d\) with finite variance. Assume: 1. mass is conserved: \[ \|u(t)\|_{L^2(\mathbb{R}^d)}=\|u_0\|_{L^2(\mathbb{R}^d)}=:M>0, \] 2. energy is conserved and negative: \[ E(u(t))=E(u_0)<0, \] 3. the variance \[ V(t)=\int_{\mathbb{R}^d}|x|^2|u(t,x)|^2\,dx \] satisfies \[ V''(t)=16E(u_0)<0, \] 4. for every sufficiently regular \(f\), \[ \|f\|_{L^2(\mathbb{R}^d)}^2 \le \frac{2}{d}\,\|x f\|_{L^2(\mathbb{R}^d)}\,\|\nabla f\|_{L^2(\mathbb{R}^d)}. \] Then there exists \(T<\infty\) such that \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \]

Finite-time blow-up from the virial identity

Node id: n0 | Status: informal

Display label: Blow-up theorem

Informal statement:

There exists \(T<\infty\) such that \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \]

Informal proof:

Let \(T>0\) be the finite time from the variance-vanishing claim. The weighted estimate gives \[ M^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)} \] for every time of existence. Here \(M>0\) is fixed, while \(V(t)\to 0\) as \(t\uparrow T\). Therefore the right-hand side can stay at least \(M^2\) only if \[ \|\nabla u(t)\|_{L^2(\mathbb{R}^d)}\to\infty \quad\text{as } t\uparrow T. \] Thus the solution blows up in finite positive time.

The variance reaches zero in finite positive time

Node id: n1 | Status: informal

Display label: Variance hits zero

Informal statement:

There exists a finite time \(T>0\) such that \[ V(T)=0, \] and hence \[ V(t)\to 0 \quad\text{as } t\uparrow T. \]

Informal proof:

Since \(E(u_0)<0\), the virial identity gives \(V''(t)=16E(u_0)<0\), so \(V\) is strictly concave. Integrating twice yields \[ V(t)=8E(u_0)t^2+at+b \] for some constants \(a,b\). Because \(E(u_0)<0\), the coefficient of \(t^2\) is negative, so \(V(t)\to -\infty\) as \(t\to +\infty\). But by definition \[ V(t)=\int_{\mathbb{R}^d}|x|^2|u(t,x)|^2\,dx\ge 0 \] for every time on which the solution exists. Therefore \(V\) cannot stay positive for all \(t\ge 0\). By continuity there exists a finite \(T>0\) with \(V(T)=0\), equivalently \(V(t)\to 0\) as \(t\uparrow T\).

Mass and the weighted inequality force a lower bound involving \(\sqrt{V(t)}\)

Node id: n2 | Status: formal_verified

Display label: Mass-variance-gradient bound

Formalization priority: 1. Rationale: Most conservative local island: it is a direct one-step derivation from the assumed weighted inequality, mass conservation, and the definition of variance. It is technically self-contained, avoids global blow-up/existence machinery, and should formalize with low dependency overhead compared with the finite-time root argument or the concavity-to-first-zero claim.

Informal statement:

For every time of existence, \[ M^2\le \frac{2}{d}\sqrt{V(t)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)}. \]

Informal proof:

Apply the weighted inequality with \(f=u(t)\): \[ \|u(t)\|_{L^2(\mathbb{R}^d)}^2 \le \frac{2}{d}\,\|x u(t)\|_{L^2(\mathbb{R}^d)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)}. \] Mass conservation gives \(\|u(t)\|_{L^2(\mathbb{R}^d)}=M\), hence \[ M^2\le \frac{2}{d}\,\|x u(t)\|_{L^2(\mathbb{R}^d)}\,\|\nabla u(t)\|_{L^2(\mathbb{R}^d)}. \] By the definition of the variance, \[ \|x u(t)\|_{L^2(\mathbb{R}^d)}^2=V(t), \] so \(\|x u(t)\|_{L^2(\mathbb{R}^d)}=\sqrt{V(t)}\). Substituting gives the claimed inequality.

import Mathlib

theorem reducedGlassey_mass_weighted_lower_bound
    {ι : Type*} {d M : ℝ}
    (massNorm xNorm gradNorm V : ι → ℝ)
    (hMass : ∀ t, massNorm t = M)
    (hxNonneg : ∀ t, 0 ≤ xNorm t)
    (hWeighted : ∀ t, massNorm t ^ 2 ≤ ((2 : ℝ) / d) * xNorm t * gradNorm t)
    (hVariance : ∀ t, xNorm t ^ 2 = V t) :
    ∀ t, M ^ 2 ≤ ((2 : ℝ) / d) * Real.sqrt (V t) * gradNorm t := by
  intro t
  have hxEq : xNorm t = Real.sqrt (V t) := by
    rw [← hVariance t, Real.sqrt_sq_eq_abs, abs_of_nonneg (hxNonneg t)]
  calc
    M ^ 2 = massNorm t ^ 2 := by rw [← hMass t]
    _ ≤ ((2 : ℝ) / d) * xNorm t * gradNorm t := hWeighted t
    _ = ((2 : ℝ) / d) * Real.sqrt (V t) * gradNorm t := by rw [hxEq]
